# 010. 아미노산 기본 패시브 설계

## 프레임워크

### 발동 방식: 스캔 시 발동 (On-Scan Trigger)

3라운드마다 윈도우가 스캔될 때, 해당 아미노산의 효과가 **"발동(Cast)"**됨.
발동된 효과의 성질에 따라 유지 시간이 자연스럽게 결정:

| 타입 | 이름 | 지속 | 설명 |
|------|------|------|------|
| 🔵 오라 (Aura) | 상태 변화형 | 이번 페이즈(3라운드) | "이번 3라운드 동안 공격력 +20%" |
| 🟡 즉발 (Instant) | 즉시 효과형 | 즉시 소멸 | "즉시 체력 10 회복" / "적에게 데미지 15" |
| 🔴 스택 (Stack) | 누적형 | 전투 종료까지 | "적에게 '부식' 스택 1 부여 (영구 누적)" |

### 네이밍 원칙: 바이오펑크 기관 메타포

유저가 화면에서 보는 이름은 **생체 기관/돌연변이 이름** 1개뿐이다.
학술 용어(아미노산 명칭)는 툴팁 하단의 플레이버 텍스트로만 노출.

```
[메인 UI — 유저가 보는 큰 글씨]
  🔴 발화샘

[툴팁 — 마우스 호버 시 하단 작은 글씨]
  합성 아미노산: 메티오닌 (Met) | ATG | 1경로 전설
```

**왜 바이오펑크 기관인가:**
- "기폭", "관통"은 마법/기계공학 스킬 느낌 → 우리 게임은 유전자 조작 바이오펑크
- "발화샘", "기생 포자"는 이름 자체가 아트 컨셉 → 아이콘/이펙트 디자인에 직결
- 유저 머릿속에 즉시 이미지가 꽂힘 ("발화샘이 켜졌다" vs "기폭이 발동했다")

### 컬러 코딩 & 아이콘 시스템

20개를 글씨로 다 읽는 건 피곤하다. **색상 + 심볼**이 0.1초 만에 정보를 전달해야 한다.

| 역할 | 색상 | 아이콘 | 직관적 규칙 |
|------|------|--------|------------|
| **Destroy** (파괴) | 🔴 붉은색 | 이빨/송곳니 | "빨간색이 많으면 아프다" |
| **Survive** (생존) | 🟢 초록색 | 방패/껍질 | "초록이 많으면 버틴다" |
| **Order** (질서) | 🔵 푸른색 | 뇌/신경 | "파란색이 많으면 빠르다" |
| **Chaos** (혼돈) | 🟣 보라색 | 소용돌이/눈동자 | "보라색이 많으면 뒤집는다" |

유저는 12칸에 Gene을 배치할 때 글씨가 아니라 **컬러 덩어리**로 게임을 읽는다:
"화면에 빨간 아이콘 3개 켜졌네 → 이번 턴 딜 나오겠다"

### 효과 설계 원칙: 4역할 기본 방향 + 고유 트위스트

| 역할 | 기본 방향 | 직관적 규칙 |
|------|----------|------------|
| **Destroy** (파괴) | 공격/데미지 강화 | "A가 많으면 아프다" |
| **Survive** (생존) | 방어/회복 | "T가 많으면 버틴다" |
| **Order** (질서) | 제어/안정/선공 | "G가 많으면 빠르다" |
| **Chaos** (혼돈) | 반전/교란/도박 | "C가 많으면 뒤집는다" |

초보: 컬러 덩어리로 역할 어림짐작 → 무리 없이 플레이
고인물: 각 기관의 고유 트위스트를 파고들어 정밀 빌드

### 희귀도-위력 스케일링

| 경로 수 | 희귀도 | 효과 강도 | 디자인 의도 |
|---------|--------|----------|------------|
| ★ 1경로 | 전설 | 게임 체인저 | 빌드 정의급. 이 코돈을 지키기 위해 서열을 짬 |
| 2경로 | 희귀 | 강력 조건부 | 특화 빌드의 핵심 |
| 3경로 | 보통 | 중간 | 안정적 |
| 4경로 | 흔함 | 약하지만 확실 | 기본 유틸. 아무 빌드에나 쓸만함 |
| 6경로 | 최흔 | 소소 | 뜨면 좋고, 안 떠도 괜찮은 수준 |

### 밸런스 안전장치

| 장치 | 적용 대상 | 설명 |
|------|----------|------|
| 스택 상한선 (Cap) | 🔴 스택 전체 | 프레임시프트 반복 등장으로 스택 폭주 방지. 구체 수치는 밸런싱 단계에서 결정 |
| 흡혈 한도 | 흡혈 촉수(Val) | 최대 HP의 X%까지만 회복 가능 |
| 양날의 검 | 혼돈(Chaos) 그룹 | 강력한 효과에는 반드시 리스크가 따름 |

## 🔴 Destroy (파괴) — 5종

> 기본 방향: 적에게 데미지를 주거나, 내 Gene의 공격력을 강화

### ★ 발화샘

| 항목 | 내용 |
|------|------|
| 아미노산 | 메티오닌 (Met) — ATG (1경로, 전설) |
| 타입 | 🟡 즉발 |
| 효과 | **이번 페이즈 첫 타격 시 무조건 적 HP에 대량 버스트 데미지 (Gene 승패 무관)** |
| 트위스트 | "시작 코돈" — 첫 Gene이 자폭하더라도 발화샘이 터지며 적을 불태운다. 전설의 확정성 보장 |
| 시너지 | 초감각 더듬이(확정 선공) + 발화샘 = 확정 선공 + 확정 폭발 오프너 |

### 산성 혈액

| 항목 | 내용 |
|------|------|
| 아미노산 | 페닐알라닌 (Phe) — TTT, TTC (2경로, 희귀) |
| 타입 | 🟡 즉발 |
| 효과 | **현재 내 12칸 서열에 남은 A(파괴) Gene 수에 비례하여 적 HP에 확정 데미지** |
| 트위스트 | 덱 압축의 핵심. A를 많이 넣을수록 피가 진해져 데미지 증가. 후반에도 A가 많이 남으면 스케일링 유지 |
| 시너지 | 기생 포자(스택) + 산성 혈액(A비례 즉발) = 상성 무관 누적+버스트 데미지 빌드 |

### 기생 포자

| 항목 | 내용 |
|------|------|
| 아미노산 | 이소류신 (Ile) — ATA, ATT, ATC (3경로, 보통) |
| 타입 | 🔴 스택 |
| 효과 | **적에게 "포자" 스택 1 부여. 매 라운드 적 HP가 스택 수만큼 감소 (영구 누적, 상한선 있음)** |
| 트위스트 | 장기전 특화. 포자가 적의 몸속에서 증식하며 갉아먹는다. 상한선(Cap)으로 무한 폭주 방지 |
| 시너지 | 신경독 가시(스택 증폭) + 기생 포자 = 독 빌드의 핵심 콤보 |

### 천공 쐐기

| 항목 | 내용 |
|------|------|
| 아미노산 | 트레오닌 (Thr) — ACA, ACT, ACG, ACC (4경로, 흔함) |
| 타입 | 🔵 오라 |
| 효과 | **이번 페이즈 동안 적의 반응성 굳은살(피해감소)과 외골격(쉴드)을 무시하고 본체 HP를 직접 타격** |
| 트위스트 | 뼈처럼 자란 쐐기가 적의 방어층을 꿰뚫는다. 탱커 빌드 상대로 진가 발휘 |
| 시너지 | 발화샘(버스트) + 천공 쐐기(방어 무시) = 탱커 킬러 빌드 |

### 탐식의 턱

| 항목 | 내용 |
|------|------|
| 아미노산 | 라이신 (Lys) — AAA, AAG (2경로, 희귀) |
| 타입 | 🔵 오라 |
| 효과 | **이번 페이즈 동안 Gene 승리 시, 다음 Gene의 기본 공격력이 영구 증폭 (스노우볼)** |
| 트위스트 | 먹이를 먹을수록 턱이 커지는 돌연변이. 승리가 승리를 부르는 눈덩이 효과 |
| 시너지 | 발화샘(첫 폭발) → 탐식의 턱(승리 후 다음 Gene 펌핑) = 개막 러시 빌드 |

## 🟢 Survive (생존) — 5종

> 기본 방향: 내 HP를 지키거나, Gene 생존율을 높임

### 외골격

| 항목 | 내용 |
|------|------|
| 아미노산 | 시스테인 (Cys) — TGT, TGC (2경로, 희귀) |
| 타입 | 🟡 즉발 |
| 효과 | **쉴드 생성. 다음 N만큼의 HP 데미지를 흡수** |
| 트위스트 | 즉시 돋아나는 키틴질 외골격. Gene 패배로 인한 HP 손실을 선제적으로 차단 |
| 시너지 | 반응성 굳은살(피해 감소) + 외골격(쉴드) = 이중 방어 |

### 흡혈 촉수

| 항목 | 내용 |
|------|------|
| 아미노산 | 발린 (Val) — GTA, GTT, GTG, GTC (4경로, 흔함) |
| 타입 | 🔵 오라 |
| 효과 | **이번 페이즈 동안 내 Gene 승리 시 HP 회복. 잃은 HP가 많을수록 회복량 증가 (최대 HP의 X%까지만 회복 가능)** |
| 트위스트 | 위기일수록 촉수가 맹렬히 흡입한다. 풀피에선 미미, 빈사에선 대량 회복. 좀비 빌드 방지용 한도 내장 |
| 시너지 | 천공 쐐기(방어 무시) + 흡혈 촉수(승리 시 회복) = 공방 일체 |

### 재생 세포

| 항목 | 내용 |
|------|------|
| 아미노산 | 세린 (Ser) — AGT, AGC, TCA, TCT, TCG, TCC (6경로, 최흔) |
| 타입 | 🟡 즉발 |
| 효과 | **즉시 HP 소량 회복. 체력이 가득 차 있다면 회복량만큼 쉴드로 전환** |
| 트위스트 | 가장 흔한 방어형. 풀피에서도 절대 꽝이 아님. 오버힐 → 쉴드 변환으로 최흔 코돈의 밸류 보장 |
| 시너지 | 어떤 빌드든 재생 세포가 뜨면 나쁘지 않음 (범용 유틸) |

### 반응성 굳은살

| 항목 | 내용 |
|------|------|
| 아미노산 | 알라닌 (Ala) — GCA, GCT, GCG, GCC (4경로, 흔함) |
| 타입 | 🔵 오라 |
| 효과 | **이번 페이즈 동안 Gene 패배 시 받는 HP 데미지 감소. 연속 패배할수록 감소율 점점 증가** |
| 트위스트 | 맞을수록 피부가 딱딱하게 경화된다. 상성 불리 구간이 연속으로 와도 점점 단단해짐 |
| 시너지 | 폭발성 담낭(패배 시 자폭) + 반응성 굳은살(연속 패배 시 점점 단단) = "맞으면서 이기는" 탱커 |

### 공생 미생물

| 항목 | 내용 |
|------|------|
| 아미노산 | 글리신 (Gly) — GGA, GGT, GGG, GGC (4경로, 흔함) |
| 타입 | 🔵 오라 |
| 효과 | **이번 페이즈 동안 무승부 또는 동종 진화(A+A=A⁺) 성공 시 내 HP 소량 회복** |
| 트위스트 | 체내 공생균이 숙주를 돌본다. 무승부+동종 진화 양쪽에서 발동. 동종 빌드의 서포터 |
| 시너지 | 융합 점균(동종 진화 증폭) + 공생 미생물(동종 진화 회복) = 동종 빌드 서포트 패키지 |

## 🔵 Order (질서) — 5종

> 기본 방향: 전투의 흐름을 제어하거나, 아군 효과를 안정/강화

### 군집 페로몬

| 항목 | 내용 |
|------|------|
| 아미노산 | 글루탐산 (Glu) — GAA, GAG (2경로, 희귀) |
| 타입 | 🔵 오라 |
| 효과 | **이번 페이즈 동안 — 오라 효과의 지속 시간 +1턴 연장, 부여되는 스택 +1 증가** |
| 트위스트 | 화학 신호로 모든 기관을 통솔한다. 단독으로는 효과 없지만 다른 기관과 합쳐지면 시너지 폭발 |
| 시너지 | 기생 포자(스택 +1 → 2배 속도) + 군집 페로몬 = 독 빌드 가속기 |

### 신경독 가시

| 항목 | 내용 |
|------|------|
| 아미노산 | 타이로신 (Tyr) — TAT, TAC (2경로, 희귀) |

| 효과 | **적에게 "신경독" 스택 부여. 적이 받는 모든 디버프/스택 효과가 신경독 수만큼 증폭** |
| 트위스트 | 가시에 찔린 적의 신경이 과민해져 모든 독이 더 잘 퍼진다. 디버프 앰프 |
| 시너지 | 기생 포자(스택) + 신경독 가시(스택 증폭) = 후반 갈수록 치명적인 독 빌드 |

### 아드레날린 선

| 항목 | 내용 |

### 초감각 더듬이

| 항목 | 내용 |

| 시너지 | 탐식의 턱(승리 후 스노우볼) + 초감각 더듬이(확정 선공) = 확정 선공 + 스노우볼 콤보 |

### 의태 색소

| 항목 | 내용 |

## 빌드 시너지 예시

### 독 빌드 (기생 포자 + 신경독 가시 + 군집 페로몬)
```
서열: [A][T][A] [T][A][T] [G][A][A] [A][T][G]
코돈: ATA(기생 포자) / TAT(신경독 가시) / GAA(군집 페로몬) / ATG(발화샘★)

페이즈1: 포자 스택 1+1(페로몬) = 2 부여 + 신경독 스택 1+1 = 2 부여 + 발화샘 버스트
페이즈2: (시프트 후) 포자 재등장 → 스택 누적 + 신경독 증폭
→ 군집 페로몬이 스택을 가속시키고, 신경독이 증폭시켜 후반 녹아내림
```

### 선공 러시 빌드 (발화샘 + 탐식의 턱 + 초감각 더듬이)
```
서열: [A][T][G] [A][A][A] [C][C][C] [T][T][T]
코돈: ATG(발화샘★) / AAA(탐식의 턱) / CCC(초감각 더듬이) / TTT(산성 혈액)

페이즈1: 초감각 더듬이(확정 선공) + 발화샘(무조건 버스트) + 탐식의 턱(승리 시 다음 Gene 펌핑) + 산성 혈액(A 비례 확정 딜)
→ 확정 선공 → 확정 폭발 → 승리 스노우볼 → A 비례 추가 딜
```

### 자폭 탱커 빌드 (반응성 굳은살 + 폭발성 담낭 + 재생 세포)
```
서열: [G][C][A] [C][A][A] [T][C][T] [G][G][G]
코돈: GCA(반응성 굳은살) / CAA(폭발성 담낭) / TCT(재생 세포) / GGG(공생 미생물)

페이즈1: 굳은살(연속 패배 시 점점 단단) + 담낭(패배 시 자폭 데미지) + 재생 세포(회복/오버힐→쉴드) + 공생 미생물(무승부 시 회복)
→ 강한 A Gene을 일부러 카운터에 배치 → 죽으면서 담낭 터짐 → 굳은살로 HP 보전 → 재생으로 회복
```

### 동종 올인 빌드 (융합 점균 + 공생 미생물 + 탐식의 턱)
```
서열: [A][A][A] [G][A][T] [A][A][C] [A][A][G]
코돈: AAA(탐식의 턱) / GAT(융합 점균) / AAC(잠복성 바이러스) / AAG(탐식의 턱)

페이즈1: A 동종 진화 연속 → 점균(옵션 한계 돌파: 하위Gene 3개 병합!) + 공생 미생물(진화 시 회복) + 탐식의 턱(승리 후 펌핑)
→ A를 몰아넣고 동종 진화로 슈퍼 A⁺(3옵션!) 양성 + 잠복성 바이러스로 생존 Gene 장기 투자
```
