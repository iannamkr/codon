# 015. 플라스미드 역설계 (Plasmid Redesign)

## 1. 역설계 원칙

### 플라스미드 = 규칙 파괴형 (Rule Breaker)

- **숫자 버프가 아님.** 숫자 보너스는 하위Gene과 아미노산이 담당.
- 모든 플라스미드는 **"X를 포기 → Y를 획득"** 구조.
- 포기하는 것의 고통이 클수록, 획득하는 것의 쾌감이 큼.
- 빌드당 1개, 전투 중 불변, 연구실에서 교체 가능.

### 역설계 근거

코어 룰(010~014)이 전부 확정된 상태에서 역설계:
- 010: 아미노산 패시브 20종
- 011: 효소 26종
- 012: 전투 공식 (HP 듀얼 트랙, 진화 ×1.5, 꼬리 귀환, 탈진)
- 013: 변이 시스템 (5등급 변이 풀, 키메라)
- 014: 체질 (6종 계수 변경)

→ 각 플라스미드는 위 시스템의 **특정 규칙 1개를 파괴**하도록 설계.

## 3. 12종 플라스미드 상세

### ═══ 전투 규칙 파괴 (3종) ═══

### #1. 역행 (Reversal)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R1 — 스캔 방향 |
| 포기 | 정방향 스캔. 배치를 처음부터 역순으로 설계해야 함 |
| 획득 | **서열을 뒤→앞(11→0)으로 스캔.** 프레임시프트도 역방향(뒤에서 당겨짐) |

```
일반: [G1][G2][G3] [G4][G5][G6] ... [G10][G11][G12]
      ──윈도우1──→ ──윈도우2──→

역행: [G1][G2][G3] ... [G10][G11][G12]
                        ←──윈도우1── ←──윈도우2──
```

- 족보(고정 윈도우 4코돈)도 역순 판정 → 완전히 다른 족보 발동
- 상대가 "앞을 강하게" 배치하는 메타를 완벽 카운터
- 뇌를 뒤집어서 빌드해야 하는 고인물 전용

---

### #2. 과성장 (Overgrowth)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R2 — 윈도우 크기 |
| 포기 | **기본 데미지(10)가 0이 됨.** 순수 아미노산 효과 + 하위Gene 보정으로만 데미지 |
| 획득 | **윈도우 3칸→4칸(Tetrad).** 4번째 Gene이 앞 3칸 코돈과 같은 역할군(색상)이면, 아미노산 패시브 효과 수치 ×3 |

```
일반: [A][T][G] = ATG(기폭) → 기본 데미지 10 + 기폭 효과
      ─── 3칸 ───

과성장: [A][T][G][A] = ATG(기폭) + 4번째 A = 🔴Destroy!
        ─── 4칸 ────
        기본 데미지 0 + 기폭 효과 ×3 = 순수 스킬 데미지만으로 승부!

        [A][T][G][T] = ATG(기폭) + 4번째 T = 🟢Survive (불일치)
        기본 데미지 0 + 기폭 효과 ×1 = 증폭 실패, 깡 데미지도 없음 → 망
```

- 4번째 Gene 색상 매칭이 핵심 퍼즐 = 서열 배치 난이도 ↑↑
- 매칭 실패 시 기본 데미지 0 + 효과 ×1 = 거의 무 데미지
- 매칭 성공 시 "발화샘 데미지 60!" 같은 미친 스킬 데미지 = 극단적 마법사 빌드
- 빌드 난이도 최상급. 보상도 최상급.

### #3. 무속성 (Null-Type)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R3a — 상성 존재 |
| 포기 | A>T>G>C>A 상성 승패 판정 (상성 유불리 소멸) |
| 획득 | **모든 1:1 대결이 ATK 비교로 판정** + 기본 데미지 +5 |

```
일반: 내 A vs 적 T → A>T 상성 승리 (ATK 무관)
무속성: 내 A(ATK 12) vs 적 T(ATK 10) → ATK 비교 12>10 → 내 승리

일반: 내 T vs 적 A → T<A 상성 패배 (ATK 무관)
무속성: 내 T(ATK 15) vs 적 A(ATK 10) → ATK 비교 15>10 → 내 승리!
```

- 상성 무시 = 타입 배치에 신경 쓸 필요 없음 → 순수 하위Gene ATK 올인
- 기본 데미지 15(10+5) = 높은 기본 화력
- 상대의 상성 카운터 배치가 완전 무의미
- 약점: 동종 진화가 빈번해짐 (ATK 같으면 동종 판정 = 진화)

### ═══ 서열 규칙 파괴 (3종) ═══

### #4. 과부하 (Overload)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R9 — 서열 크기 |
| 포기 | **탈진 데미지 ×2** (순환 가속 → 장기전 불리) |
| 획득 | **서열 12칸 → 16칸** (Gene +4) |

```
일반:   [12칸] + 탈진 5/10/15/20...
과부하: [16칸] + 탈진 10/20/30/40...
```

- Gene 4개 추가 = 코돈 조합 가능성 확대 + 전소 내구도 ↑
- 탈진 ×2 = 순환 2회째에 이미 누적 30 → 3~4회전이면 사망
- 아이러니: Gene은 많지만 시간은 짧음 → "초반에 많은 Gene으로 밀어붙여라"
- 과부하 + 맹공 체질 = "16Gene 러시" 아키타입

### #5. 역류 (Reflux)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R5 — 꼬리 귀환 방향 |
| 포기 | **생존 Gene이 머리로 돌아올 때마다 본체 최대 HP -2 영구 감소** |
| 획득 | **생존 Gene이 서열 맨 앞(Head)으로 귀환** + 프레임시프트 방향 반전(앞→뒤 당겨짐) |

```
일반: [G1] 승리 → Survivor Pool → 서열 끝으로
      [G2][G3]...[G12][G1]
                        ↑ 뒤에서 천천히 순환

역류: [G1] 승리 → Survivor Pool → 서열 맨 앞으로
      [G1][G2][G3]...[G12]
       ↑ 바로 다음 턴에 재등장! 단, 최대 HP -2
```

- 에이스가 계속 선봉에 서는 "무한 에이스" 빌드
- 대가: 귀환할 때마다 내 목숨을 갉아먹음 = 진정한 역류(구토)
- 5회 귀환 = HP -10. 10회 귀환 = HP -20. 장기전일수록 치명적
- 시너지: 재생 체질(높은 HP) + 재생 세포(회복) → HP 감소를 버티면서 에이스 재활용
- 시너지: 맹공 체질 → 에이스가 빠르게 적 전소시키고 HP 감소 전에 끝냄

---

### #6. 재조합 (Recombination)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R10 — 잠김 규칙 |
| 포기 | **덮어쓸 때마다 최대 HP 영구 -5** |
| 획득 | **연구실에서 잠긴(★) Gene을 새 Gene으로 교체 가능** |

```
일반: [강타A★][보호T★][가속G] — ★ Gene은 수정 불가
재조합: [강타A★] → 교체! [폭주C] (HP -5)
        [보호T★] → 교체! [정밀G] (HP -5, 누적 -10)
```

- 결정화(잠김 누적)로 망한 실험체를 구제하는 유일한 방법
- HP를 깎아 수명을 연장 = "재조합 수술"
- 교체 횟수에 자연 한계: HP가 0에 가까워지면 더 이상 교체 불가
- 세대 교체를 최대한 지연시키는 "영생(延生)" 플레이
- 시너지: 재생/철벽 체질(높은 HP) → 교체 여유 확보

### ═══ 변이 규칙 파괴 (3종) ═══

### #7. 불안정 서열 (Unstable Sequence)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R6 — 변이 수락/거부 |
| 포기 | **변이 거부 선택권 (모든 변이 자동 수락)** |
| 획득 | **변이 확률 2배** (분모 50) + **선택지 항상 +1개** |

```
일반: 변이 팝업 → [수락] [거부]
불안정: 변이 팝업 → 자동 수락! (선택지 2~3개 중 최선을 시스템이 자동 선택... 이 아니라 유저가 선택)

정확한 동작:
  1. 변이 발동 (확률 2배)
  2. 선택지 N+1개 표시
  3. 유저가 "어떤 변이"를 적용할지 선택 (이건 가능)
  4. 선택한 변이가 자동 수락됨 (거부 불가)
```

- "변이 자체는 피할 수 없지만, 어떤 변이인지는 고를 수 있다"
- 카오스 빌드의 극치. 전투마다 서열이 완전히 다른 모양으로 변형
- 시너지: 변이 체질(MUT 분모 70→불안정 서열 50) → 미친 변이 빈도

### #8. 순수 서열 (Pure Sequence)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R8 — 변이 존재 |
| 포기 | **변이 시스템 전체 비활성화** (MUT 스탯 사실상 사망) |
| 획득 | **전 스탯 파생 +20%** (체질 계수에 ×1.2 적용) |

```
순수 서열 + 맹공 체질:
  기본 데미지 = 10 + STR × 0.18 × 1.2 = 10 + STR × 0.216
  HP = 80 + STR × 1.5 × 1.2 = 80 + STR × 1.8

STR 40:
  데미지: 10 + 8.64 = 18.64 (맹공 기본 17.2, +8.4%)
  HP: 80 + 72 = 152 (맹공 기본 140, +8.6%)
```

- 변이 팝업 없음, MUT 무의미 → 연구실에서 깎은 빌드가 전투 끝까지 완벽 보존
- 안정형 + 올라운드 스탯 부스트
- "변이 따위 필요 없다. 내 빌드가 완벽하니까."

### #9. 적응체 (Adaptive)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R7 — 변이 영구성 |
| 포기 | **변이 효과 ×0.5** |
| 획득 | **변이 비영구** — 수락해도 전투 중만 적용, 귀환 시 원복. **잠김도 발생 안 함** |

```
적응체 + 점변이:
  전투 중: G₃ → C₃ 변환 (효과 ×0.5 = 코돈 효과 절반으로 발동)
  귀환 후: G₃ 원복. 잠김 없음.
```

- 변이를 "시험 삼아" 사용 → 좋으면 다음 빌드에 반영, 나쁘면 무시
- 빌드 오염 완전 방지. 결정화 리스크 제로
- 효과 절반이라는 대가 = 변이의 폭발적 쾌감은 떨어짐

### #10. 반전체 (Inverter)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R3b — 상성 방향 |
| 포기 | **정상 상성 방향 (내 빌드도 역상성 기준으로 짜야 함)** |
| 획득 | **상성 완전 역전: A<T<G<C<A** |

```
정상: A>T>G>C>A
반전: A<T<G<C<A (= T>A, G>T, C>G, A>C)

정상 메타: A를 앞에 배치하여 T를 잡는다
반전체:   T를 앞에 배치하여 A를 잡는다 → 메타 완벽 카운터
```

- 상대의 "정상 메타" 배치를 카운터하는 궁극의 안티메타
- 단, 내 빌드도 역상성 기준으로 완전히 재설계해야 함
- 반전체 vs 반전체 = 정상 상성과 동일 (역전의 역전 = 원래)
- 반전체 vs 무속성 = 무속성 우세 (상성 자체가 없으므로 역전 무의미)

### #11. 잠복 (Dormancy)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R11 — 전투 타이밍 |
| 포기 | **1~2페이즈(6라운드) 동안 내 모든 아미노산 패시브 + 효소 효과 완전 비활성화.** 순수 깡스탯으로만 쳐맞으며 버텨야 함 |
| 획득 | **3페이즈 시작 시, 비활성화되었던 모든 아미노산 효과가 한 번에 중첩 폭발 발동** |

```
═══ Phase 1~2 (동면기) ═══
아미노산 패시브: 비활성
효소: 비활성
전투: 순수 상성 + 기본 데미지(10) + 하위Gene 보정만으로 싸움
→ 피투성이가 되며 버팀. 상대는 풀 파워로 공격 중.

═══ Phase 3 시작 (각성!) ═══
"동면 해제! 축적된 아미노산 효과 일괄 발동!"

축적된 아미노산 목록:
  - 발화샘(Phase 1 코돈): 치명타 발동!
  - 외골격(Phase 1 코돈): 쉴드 생성!
  - 탐식의 턱(Phase 2 코돈): 영구 펌핑!
  - 군집 페로몬(Phase 2 코돈): 아군 버프!
  → 4개 아미노산 동시 폭발! = "원펀맨 각성"

═══ Phase 3~4 (풀 파워) ═══
이후 정상 전투 + 축적 버프 유지
```

- 초반 6라운드를 피투성이로 버티는 인내의 빌드
- 3페이즈 각성 시 모아둔 버프가 한 번에 터지는 카타르시스
- 시너지: 철벽/재생 체질(높은 HP로 동면기 생존) + 반응성 굳은살(연속 패배 감소)
- 리스크: 동면기에 Gene 전소되면 각성 못하고 패배
- 방송용 하이라이트: "6턴 동안 쳐맞더니... 7턴째에 화면이 온통 이펙트로 뒤덮였습니다!"

### #12. 기생체 (Parasite)

| 항목 | 내용 |
|------|------|
| 파괴 규칙 | R12 — 상대 플라스미드 |
| 포기 | **자체 플라스미드 능력 없음** (기생체 단독 = 효과 0) |
| 획득 | **상대 플라스미드를 무효화 + 그 효과를 내가 사용** |

```
기생체 vs 잠복:
  상대의 잠복 비활성화 + 내가 잠복 효과 사용
  → 내가 동면기 → 각성 폭발. 상대는 정상 전투.

기생체 vs 순수 서열:
  상대의 순수 서열 비활성화 + 내가 파생 +20% 획득
  → 상대는 변이 시스템 정상 적용 (순수 서열 해제됨)

기생체 vs 기생체:
  양쪽 모두 탈취 대상 없음 → 양쪽 플라스미드 없는 상태로 전투
```

- 궁극의 메타 카운터. "네가 뭘 쓰든 나도 그걸 쓴다"
- 약점: 상대가 플라스미드 없이 싸우는 것보다, 내가 상대 플라스미드를 훔치는 이득이 클 때만 유효
- 기생체 vs 기생체 = 가장 공정한 전투 (둘 다 깡 빌드)

## 4. 상성 매트릭스 (플라스미드 vs 플라스미드)

주요 플라스미드 간 상호작용:

| | 무속성 | 반전체 | 잠복 | 기생체 |
|---|--------|-------|------|--------|
| **무속성** | ATK 싸움 | 무속성 유리(역전 무의미) | 잠복 동면기에 ATK로 밀어붙임 | 기생체가 데미지+5 획득 |
| **반전체** | 불리(역전 무효) | 정상 상성(역전의 역전) | 잠복 동면기를 역상성으로 공략 | 기생체가 역전 획득 |
| **잠복** | 동면기 피해 큼 | 역상성에 동면기 취약 | 양쪽 동면→양쪽 각성 | 기생체가 잠복 탈취 |
| **기생체** | +5 데미지 획득 | 역전 획득 | 잠복 탈취 | 양쪽 플라스미드 없음 |

## 5. 빌드 아키타입 예시

### "극딜 마법사" (과성장 + 맹공 + 발화샘)
```
플라스미드: 과성장 (기본 데미지 0, 4칸 매칭 시 아미노산 ×3)
체질: 맹공 (데미지 계수 ↑)
서열: 모든 윈도우의 4번째 Gene을 🔴Destroy로 배치
효소: A-스테로이드 앰플 + 피의 펌프 모터

→ 발화샘(크리) ×3 = 60%+ 크리 확률
→ 기본 데미지 0이지만, 크리 시 아미노산 스킬 데미지만으로 30+ 히트
→ 4번째 Gene 매칭 실패 시 데미지 0 → 극단적 올인
```

### "무한 에이스" (역류 + 재생 + 흡혈 촉수)
```
플라스미드: 역류 (머리 귀환, HP -2/회)
체질: 재생 (HP 성장 +25%)
서열: 1번 슬롯에 최강 A Gene (ATK 최대), 뒤에 회복 서포트
효소: 극저온 동결 캡슐 + T-항체 배양액

→ A₁이 승리 → 머리 귀환 → 바로 재등장 → 또 승리 → ...
→ HP -2씩 깎이지만 재생 체질 HP 180+ → 90회 귀환 가능 (사실상 무한)
→ 흡혈 촉수(승리 시 HP 회복)로 HP 감소 상쇄
```

### "원펀맨" (잠복 + 철벽 + 반응성 굳은살)
```
플라스미드: 잠복 (6라운드 동면 → 각성 폭발)
체질: 철벽 (HP/DEF 높음, 동면기 생존)
서열: 앞 6칸에 Survive+Order 코돈 (동면기 깡 생존용), 뒤 6칸에 Destroy 폭탄
효소: 경화 수지 스프레이 + 충격 흡수 겔 + 농축 발화 촉매제 + 한계 돌파 자극기

→ Phase 1~2: DEF 45% + 기본 데미지 8로 맞으며 버팀
→ Phase 3: 축적된 외골격+재생 세포+발화샘+탐식의 턱 일괄 폭발!
→ 방송 하이라이트: "6턴 버티더니 7턴째에 다 죽었습니다"
```

### "절대 카운터" (기생체 + 변이 체질)
```
플라스미드: 기생체 (상대 플라스미드 탈취)
체질: 변이 (카오스 빌드 기본값)
서열: 범용 배치 (상대 플라스미드에 따라 유동)

→ 상대가 과성장? 나도 과성장 (상대는 일반 3칸으로 전락)
→ 상대가 순수 서열? 나도 파생 +20% (상대는 변이 활성화)
→ 상대도 기생체? 깡 빌드 대결 → 변이 체질의 높은 MUT이 차이를 만듦
```
