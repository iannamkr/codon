# 011. 효소(조커) 효과 설계

## 프레임워크

### 정체성: 연구원의 도구

효소는 **연구원(유저)이 실험체에 주입하는 약물/장비/도구**다.
아미노산 패시브(실험체의 몸에서 자라나는 기관)와 레이어가 분리됨:

| 레이어 | 무엇 | 비유 | 통제 |
|--------|------|------|------|
| 아미노산 패시브 | 실험체의 몸 (기관/돌연변이) | "이 괴물은 이렇게 자랐다" | 간접 (Gene 배치로 코돈 유도) |
| **효소** | 연구원의 개입 (약물/도구) | "나는 이 약을 주입한다" | **직접 (4슬롯에 장착)** |

### UI 표시 원칙

```
[메인 UI — 효소 슬롯 (상단 4칸)]
  🔴 농축 발화 촉매제    🧬 패턴 인식

[툴팁 — 마우스 호버]
  파괴 그룹 아미노산(발화샘, 산성 혈액 등) 발동 시
  이번 페이즈 동안 30% 확률로 치명타(데미지 2배)
```

효소 카드에는 **역할 색상 라벨**(🔴🟢🔵🟣💀) + **트리거 아이콘**(🧬🧪⚡) 표시.
유저는 슬롯 4개의 색상 조합만 봐도 빌드 방향을 직감.

### 슬롯 규칙

- **4슬롯 장착**, 연구실에서 세팅, 전투 중 불변
- **전투 후 자유 교체** (Gene만 잠김, 효소는 잠기지 않음)
- **중복 장착 불가** — 같은 효소를 2개 이상 넣을 수 없음
- 프레임시프트 후 3라운드마다 효소도 **재판정** (조건 변화 감지)

### 트리거 카테고리

| 아이콘 | 카테고리 | 트리거 시점 | 설명 |
|--------|----------|------------|------|
| 🧬 | **패턴 인식** | 아미노산 패시브 발동 후 | 특정 아미노산(기관)이 발동했을 때 시너지 증폭 |
| 🧪 | **조건 인식** | 윈도우 스캔 시 | 3칸 내 Gene 구성, HP 상태, 서열 상태 등을 스캔 |
| ⚡ | **이벤트 인식** | 전투/시프트 결과 시 | Gene 승패, 프레임시프트 발생, 페이즈 전환에 반응 |

### 작동 순서 (1페이즈 내)

```
1. [스캔]      시스템이 현재 윈도우(3칸) 스캔 → 코돈 패턴 판정
2. [아미노산]   코돈의 기본 패시브 발동 (010 문서의 20종)
3. [효소 개입]  장착한 4개 효소가 순차 체크:
               🧬 패턴형 → 발동된 아미노산에 반응
               🧪 조건형 → 윈도우/상태를 스캔
4. [전투]      3라운드 동안 Gene 1:1 전투
               ⚡ 이벤트형 → 승패/시프트 결과에 반응
5. [시프트]    소멸/생존 처리 → 프레임시프트 → 다음 윈도우 스캔
```

### 설계 매트릭스: 4역할 × 3트리거 × 2종 + Stop 2종

| | 🧬 패턴 | 🧪 조건 | ⚡ 이벤트 | 소계 |
|---|---------|---------|----------|------|
| 🔴 Destroy | 2 | 2 | 2 | 6 |
| 🟢 Survive | 2 | 2 | 2 | 6 |
| 🔵 Order | 2 | 2 | 2 | 6 |
| 🟣 Chaos | 2 | 2 | 2 | 6 |
| 💀 Stop | — | 1 | 1 | 2 |
| **합계** | **8** | **9** | **9** | **26** |

## 🔴 Destroy — 파괴 유발 도구 (6종)

> 공격의 한계를 돌파하는 전투용 약물

### 🧬 농축 발화 촉매제

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 파괴 그룹 아미노산 발동 시 |
| 효과 | **파괴 그룹 아미노산(발화샘, 산성 혈액, 기생 포자, 천공 쐐기, 탐식의 턱)이 발동하면, 이번 페이즈 동안 내 공격이 30% 확률로 치명타(데미지 2배)로 적중** |
| 컨셉 | 발화샘에 기름을 부어 폭발력을 끌어올리는 촉매제. 파괴 코돈이 많을수록 치명타 찬스가 많아짐 |
| 시너지 | 발화샘(첫 타격 버스트) + 촉매제(치명타) = 오프닝 원킬 |

### 🧬 골수 추출 피펫

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 탐식의 턱 또는 천공 쐐기 발동 중 |
| 효과 | **탐식의 턱이나 천공 쐐기가 발동 중일 때 적 Gene을 파괴하면, 적 본체 최대 HP의 5%를 추가로 뜯어냄** |
| 컨셉 | 파괴한 적 Gene에서 골수를 빨아내는 주사기. HP가 높은 적일수록 많이 빼앗김 — 체력 돼지 카운터 |
| 시너지 | 탐식의 턱(승리 스노우볼) + 피펫(%기반 추출) = 보스 킬러 |

### 🧪 A-스테로이드 앰플

| 항목 | 내용 |
|------|------|
| 트리거 | 🧪 조건 — 윈도우 3칸 내 A가 2개 이상 |
| 효과 | **이번 페이즈 동안 내 모든 Gene 기본 공격력 ×1.5 증폭** |
| 컨셉 | A(파괴) Gene에 직접 주입하는 근육 증강제. A를 많이 배치할수록 보상 |
| 시너지 | 산성 혈액(A 수 비례 확정 딜) + 앰플(A 밀집 공격 증폭) = A 도배 빌드 |

### 🧪 한계 돌파 자극기

| 항목 | 내용 |
|------|------|
| 트리거 | 🧪 조건 — 전체 서열(12칸) 중 살아있는 A가 5개 이상 |
| 효과 | **상시 발동. 내 모든 타격이 적의 쉴드(외골격)를 관통** |
| 컨셉 | A Gene 도배를 극한까지 밀어붙인 덱에 대한 보상. 방어 무시로 공격에만 올인 |
| 시너지 | 천공 쐐기(방어 관통) + 자극기(쉴드 관통) = 완전 관통 빌드 |

### ⚡ 피의 펌프 모터

| 항목 | 내용 |
|------|------|
| 트리거 | ⚡ 이벤트 — 내 Gene이 적 Gene을 파괴할 때마다 |
| 효과 | **남은 전투 동안 내 모든 공격력 +2 영구 누적** |
| 컨셉 | 킬을 먹을 때마다 모터가 가속. 전투가 길어질수록 딜이 눈덩이처럼 불어남 |
| 시너지 | 탐식의 턱(승리 시 다음 Gene 펌핑) + 모터(킬당 영구 +2) = 이중 스노우볼 |

### ⚡ CRISPR 절단 가위

| 항목 | 내용 |
|------|------|
| 트리거 | ⚡ 이벤트 — 프레임시프트(서열 당겨짐) 발생 시 |
| 효과 | **다음 적이 두르고 있는 쉴드와 버프를 즉시 제거** |
| 컨셉 | 서열이 당겨지는 순간 가위가 적의 방어를 잘라냄. 프레임시프트를 공격 기회로 전환 |
| 시너지 | 기생 포자(프레임시프트로 재등장) + 가위(시프트 시 적 버프 제거) = 시프트 빌드 |

## 🟢 Survive — 생존 보조 도구 (6종)

> 어떻게든 멱살 잡고 살려내는 의료 장비

### 🧬 경화 수지 스프레이

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 생존 그룹 아미노산 발동 시 |
| 효과 | **생존 그룹 아미노산(외골격, 흡혈 촉수, 재생 세포, 반응성 굳은살, 공생 미생물)이 발동하면, 얻는 쉴드량/방어 수치 ×1.5 증폭** |
| 컨셉 | 방어 기관에 경화 수지를 뿌려 두께를 늘리는 스프레이. 생존 코돈이 많을수록 철벽 |
| 시너지 | 외골격(쉴드 생성) + 스프레이(쉴드 ×1.5) = 초대형 쉴드 |

### 🧬 세포 분열 억제제

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 재생 세포 또는 공생 미생물 발동 시 |
| 효과 | **현재 잃은 HP가 많을수록 회복량 최대 ×3까지 증폭** |
| 컨셉 | 세포 분열을 억제했다가 한꺼번에 터뜨리는 약물. 빈사 상태에서 대량 회복 |
| 시너지 | 재생 세포(즉시 회복) + 억제제(저체력 시 ×3) = 역전 힐 |

### 🧪 T-항체 배양액

| 항목 | 내용 |
|------|------|
| 트리거 | 🧪 조건 — 윈도우 3칸 내 T가 2개 이상 |
| 효과 | **페이즈 시작 시 내게 걸린 디버프 스택(기생 포자, 신경독 등)을 1개 정화** |
| 컨셉 | T(생존) Gene이 모이면 항체가 생성되어 독을 중화. 독 빌드 카운터 |
| 시너지 | 반응성 굳은살(피해 감소) + 항체(디버프 정화) = 독 면역 탱커 |

### 🧪 극저온 동결 캡슐

| 항목 | 내용 |
|------|------|
| 트리거 | 🧪 조건 — 내 본체 HP가 30% 이하 |
| 효과 | **상시 발동. 받는 모든 데미지 50% 강제 감소** |
| 컨셉 | 체온이 떨어지면 자동으로 냉동 보호막이 전개. 죽기 직전에 가장 단단해짐 |
| 시너지 | 세포 분열 억제제(저체력 회복 ×3) + 동결 캡슐(저체력 데미지 -50%) = 좀비 빌드 코어 |

### ⚡ 충격 흡수 겔

| 항목 | 내용 |
|------|------|
| 트리거 | ⚡ 이벤트 — 1:1 전투에서 무승부(동등) 발생 시 |
| 효과 | **겔이 팽창하며 내게 쉴드 +15 부여** |
| 컨셉 | 충격이 상쇄될 때 충격파를 흡수하여 방어막으로 전환 |
| 시너지 | 의태 색소(불리→무승부 전환) + 겔(무승부 시 쉴드) = 무승부 양산 탱커 |

### ⚡ 제세동기

| 항목 | 내용 |
|------|------|
| 트리거 | ⚡ 이벤트 — 상성 불리로 내 Gene이 파괴될 때 |
| 효과 | **20% 확률로 전기 충격, 해당 Gene을 HP 1로 부활 (한 턴 더 생존)** |
| 컨셉 | 심장이 멈춘 Gene에 전기를 쏴서 억지로 살려냄. 한 번 더 싸울 기회 |
| 시너지 | 폭발성 담낭(파괴 시 자폭 데미지) + 제세동기(부활) = 자폭 후 부활하여 재자폭 |

## 🔵 Order — 질서 통제 도구 (6종)

> 전투의 판과 룰을 내 마음대로 조작하는 해킹 장치

### 🧬 신경망 오버라이드 칩

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 질서 그룹 아미노산 발동 시 |
| 효과 | **이번 페이즈 내내 내 상성 불리 패널티를 완전히 무효화** |
| 컨셉 | 신경 회로를 덮어써서 "약점"이라는 개념 자체를 삭제. 질서(Order)의 극한 — 룰을 재정의 |
| 시너지 | 의태 색소(불리→무승부) + 오버라이드(불리 무효화) = 상성 면역 |

### 🧬 위상 교정 렌즈

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 의태 색소 발동 중 |
| 효과 | **의태 색소가 불리→무승부로 전환할 때, 무승부가 아니라 상성 유리 판정으로 역전** |
| 컨셉 | 의태 색소의 위장을 더욱 정교하게 보정하는 렌즈. 불리가 오히려 유리로 뒤집힘 |
| 시너지 | 의태 색소(불리→무승부) + 렌즈(무승부→유리) = 불리가 유리로 역전 |

### 🧪 G-시냅스 연결선

| 항목 | 내용 |
|------|------|
| 트리거 | 🧪 조건 — 윈도우 3칸 내 G가 2개 이상 |
| 효과 | **이번 페이즈 동안 내 SPD가 무조건 적보다 높아짐 (확정 선공)** |
| 컨셉 | G(질서) Gene들이 시냅스로 연결되어 반사 속도가 극대화. 확정 선공 보장 |
| 시너지 | 초감각 더듬이(확정 선공) + 시냅스(G 밀집 확정 선공) = 이중 선공 보장 |

### 🧪 DNA 시퀀스 스캐너

| 항목 | 내용 |
|------|------|
| 트리거 | 🧪 조건 — 윈도우 3칸 내 A/T/G/C 중 3종류 이상 |
| 효과 | **이번 페이즈 동안 공격력과 방어력이 동시에 ×2 증가** |
| 컨셉 | 다양한 Gene이 모이면 스캐너가 완전한 시퀀스를 인식하여 전체 성능 부스트. 무지성 도배 방지, 퍼즐 보상 |
| 시너지 | 군집 페로몬(오라 +1턴/스택 +1) + 스캐너(다양성 보상) = 범용 밸런스 빌드 |

### ⚡ 리듬 조율 마이크로칩

| 항목 | 내용 |
|------|------|
| 트리거 | ⚡ 이벤트 — 새 페이즈(3라운드) 시작 시 |
| 효과 | **첫 타격의 상성을 무조건 무승부(동등) 처리** |
| 컨셉 | 페이즈 전환 시 리듬을 재조율하여 첫 교전의 위험을 제거. 오프닝 돌연사 방지 |
| 시너지 | 충격 흡수 겔(무승부 시 쉴드) + 조율칩(첫 타 무승부) = 매 페이즈 안전한 시작 |

### ⚡ 형태 유지 프레임

| 항목 | 내용 |
|------|------|
| 트리거 | ⚡ 이벤트 — 내 Gene이 승리하여 Survivor Pool로 이동할 때 |
| 효과 | **해당 Gene에 남은 체력만큼을 즉시 내 본체 쉴드로 전환** |
| 컨셉 | 생존한 Gene의 잔여 에너지를 프레임이 추출하여 본체를 보호 |
| 시너지 | 잠복성 바이러스(생존 Gene 강화) + 프레임(생존 Gene의 HP→쉴드) = 생존 Gene 극대화 |

## 🟣 Chaos — 혼돈 조장 도구 (6종)

> 실험체를 괴롭혀서 억지 시너지를 짜내는 약물

### 🧬 돌연변이 안정제

| 항목 | 내용 |
|------|------|
| 트리거 | 🧬 패턴 — 혼돈 그룹 아미노산 발동 시 |

| 시너지 | 망각의 플라시보(Stop 시 버프 유지) + 배양액(Stop 시 20% HP 고정 딜) = Stop 코돈이 최강의 무기가 됨 |

## 빌드 아키타입 예시

### 🔴 A-도배 버서커 (순수 파괴)
```
효소: [A-스테로이드 앰플] + [한계 돌파 자극기] + [피의 펌프 모터] + [농축 발화 촉매제]
서열: A를 8~10칸 배치, 나머지 T/G로 최소 생존

전략: A 밀집으로 앰플(×1.5)+자극기(쉴드 관통) 상시 발동
      킬마다 모터(+2) 누적, 파괴 아미노산 발동 시 촉매제(30% 치명타)
      → 적이 뭘 해도 뚫는 브루트포스 빌드
```

### 🟢 좀비 탱커 (불사 생존)
```
효소: [극저온 동결 캡슐] + [세포 분열 억제제] + [경화 수지 스프레이] + [충격 흡수 겔]
서열: T를 6~8칸, G로 무승부 유도

전략: HP 30% 이하 → 동결(데미지 -50%) + 억제제(회복 ×3) = 죽지 않음
      생존 아미노산 발동 시 스프레이(방어 ×1.5)
      무승부마다 겔(쉴드 +15) = 의태 색소와 시너지
      → 때려도 안 죽고, 때릴수록 단단해짐
```

### 🟣 네크로맨서 (괴사 빌드)
```
효소: [괴사 조직 배양액] + [망각의 플라시보] + [사체 방부 처리제] + [폐기물 재활용 분쇄기]
서열: 의도적으로 Stop 코돈이 만들어지도록 TAA/TAG/TGA 패턴 배치

전략: Stop 코돈 밟을 때마다 → 배양액(적 HP 20%) + 플라시보(버프 유지)
      방부 처리제로 잠긴 Gene 해제 = 결정화 지연
      분쇄기로 남은 잠긴 Gene에서 데미지 보너스 흡수
      → "함정을 무기로, 노화를 연료로" 쓰는 하드코어 빌드
```

### 🔵 퍼즐 마스터 (질서 제어)
```
효소: [DNA 시퀀스 스캐너] + [위상 교정 렌즈] + [신경망 오버라이드 칩] + [형태 유지 프레임]
서열: A/T/G/C를 3:3:3:3으로 균등 배치

전략: 스캐너(3종 이상 → 공방 ×2) + 오버라이드(불리 무효) + 렌즈(불리→유리)
      승리한 Gene의 HP가 프레임으로 쉴드 전환
      → 상성을 완전히 무시하고 퍼즐처럼 최적 배치를 찾는 빌드
```

### 🟣💀 자폭 카운터 (역발상)
```
효소: [불안정 동위원소] + [폭발성 담낭(아미노산)] + [제세동기] + [돌연변이 안정제]
서열: 높은 공격력 A Gene을 일부러 카운터 위치에 배치

전략: Gene이 죽으면 → 담낭(자폭) + 동위원소(적 공격력 ×200% 반사) = 적이 강할수록 반사 극대화
      제세동기(20% 부활) → 부활하면 다시 죽어서 다시 자폭
      안정제로 Chaos 리스크 절반 → 리스크 대비 리턴 극대화
      → "나를 때리면 너도 죽는다" 카운터 빌드
```
