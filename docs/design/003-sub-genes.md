# Sub-Gene (하위 유전자) 설계

## 정의

하위 Gene = 각 ATGC 카테고리 내의 변종. Gene에 **내장된 보너스 효과**.

```
[강타하는 A] = 하나의 완성품 아이템
  → 베이스: A (코돈 판정에 사용)
  → 하위 효과: 강타 (전투 시 위력 배율 증가)

일체형 아이템 모델: Gene과 하위 Gene은 분리되지 않음.
인벤토리에서 줍거나 배양하는 시점에 이미 하위 효과가 결정됨.
```

- 코돈(아미노산) 기본 패시브는 베이스(A/T/G/C) 패턴으로 결정 — 하위 Gene과 무관
- 하위 Gene은 Gene 1:1 전투 시 **개별 보너스**로 적용
- 잠긴 Gene의 하위 효과 = 실험체의 영구 자산 (결정화 시너지)

## 총 40종 (카테고리당 10종)

### A (파괴) — 10종

| # | 하위 Gene | 보너스 효과 |
|---|-----------|-----------|
| 1 | **A-강타** | 위력 배율 증가 |
| 2 | **A-폭발** | 주변 Gene에 스플래시 데미지 |
| 3 | **A-관통** | 상대 DEF 일부 무시 |
| 4 | **A-침식** | 지속 피해 추가 (누적) |
| 5 | **A-약화** | 상대 스탯 디버프 |
| 6 | **A-처형** | 상대 HP 낮을수록 데미지 증가 |
| 7 | **A-희생** | 자기 HP를 소모해서 추가 데미지 |
| 8 | **A-선제** | 선공 시 위력 대폭 증폭 |
| 9 | **A-파쇄** | 상대 실드/버프를 파괴 |
| 10 | **A-과부하** | 이전 라운드에서 데미지 줬으면 위력 누적 증가 |

### T (생존) — 10종

| # | 하위 Gene | 보너스 효과 |
|---|-----------|-----------|
| 1 | **T-보호** | 받는 데미지 감소 |
| 2 | **T-회복** | HP 회복 |
| 3 | **T-반격** | 받은 데미지 일부 반사 |
| 4 | **T-흡혈** | 준 데미지의 일부를 HP로 회복 |
| 5 | **T-장벽** | 다음 피해를 흡수하는 실드 생성 |
| 6 | **T-인내** | 치명적 피해 시 HP 1로 생존 (1회) |
| 7 | **T-은신** | 다음 공격 1회 회피 |
| 8 | **T-적응** | 같은 타입 공격 반복 시 저항 증가 |
| 9 | **T-정화** | 자신에게 걸린 디버프 제거 |
| 10 | **T-공생** | HP 낮을수록 방어력 증가 |

### G (질서) — 10종

| # | 하위 Gene | 보너스 효과 |
|---|-----------|-----------|
| 1 | **G-가속** | SPD 보너스 |
| 2 | **G-정밀** | 효과 수치 증폭 |
| 3 | **G-안정** | 변이 저항 + 효과 변동 감소 |
| 4 | **G-지속** | 버프/디버프 지속시간 연장 |
| 5 | **G-집중** | 이번 약하지만 다음 라운드 효과 강화 |
| 6 | **G-억제** | 상대 버프 제거/무효화 |
| 7 | **G-공명** | 인접 Gene이 같은 ATGC면 시너지 보너스 |
| 8 | **G-환경** | 환경 효과 강화 |
| 9 | **G-결속** | 인접 Gene과 보너스 효과 공유 |
| 10 | **G-선독** | 선공권 보장 (SPD 무시) |

### C (혼돈) — 10종

| # | 하위 Gene | 보너스 효과 |
|---|-----------|-----------|
| 1 | **C-폭주** | 크리티컬 확률/배율 증가 |
| 2 | **C-전파** | 효과가 인접 Gene으로 전파 |
| 3 | **C-왜곡** | 랜덤 추가 효과 발동 |
| 4 | **C-연쇄** | 같은 효과가 2회 발동 |
| 5 | **C-도발** | 상대 다음 Gene의 행동 변경 |
| 6 | **C-복제** | 아미노산 효과를 복사해 추가 적용 |
| 7 | **C-잠식** | 상대 Gene의 보너스 효과를 훔침 |
| 8 | **C-역전** | 후공 시 오히려 위력 대폭 증가 |
| 9 | **C-기생** | 상대에게 시한폭탄 심기 (N라운드 후 폭발) |
| 10 | **C-분열** | 효과가 여러 약한 효과로 분산 (다중 타격) |
