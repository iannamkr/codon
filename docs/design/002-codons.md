# Codon (코돈) 설계

## 핵심 개념

생물학의 코돈을 그대로 차용하되, **코돈 = 패턴 + 아미노산 기본 패시브**.

```
코돈은 컨테이너(상자)가 아니라 패턴(무형)이다.
12칸 Gene 서열을 3개씩 끊어 읽으면 코돈 패턴이 형성됨.

[강타A] [보호T] [가속G] → 코돈 ATG → 아미노산 "기폭" (기본 패시브)
[폭주C] [정밀G] [왜곡C] → 코돈 CGC → 아미노산 "적응" (기본 패시브)
[흡혈T] [반격T] [폭발A] → 코돈 TTA → 아미노산 "가속" (기본 패시브)
```

- **Gene**: A, T, G, C — 4종류 물리적 블록 (하위 효과 내장)
- **코돈**: Gene 3개의 패턴 = 아미노산 기본 패시브 (4³ = 64가지)
- **아미노산**: 코돈이 결정하는 기본 패시브 효과 (20종 + 정지 3종)
- 코돈 판정 시 **베이스(A/T/G/C)만 읽음** — 하위 Gene은 무시

## 2레이어 효과 구조

```
레이어 1: 족보 (빌드 시 확정, 전투 내내 유지)
  12 Gene → 4코돈 → 4개 아미노산 패시브 = 전투 내내 고정

레이어 2: 코돈 효과 (전투 중 3라운드마다 재평가)
  프레임시프트 후 남은 Gene을 다시 3개씩 읽음 → 새 코돈 효과 발현
```

## 아미노산 기본 패시브 효과 (20종 + 정지)

> 아래는 구 시스템("스킬")의 효과 서술. 새 시스템에서는 "아미노산 기본 패시브"로 재해석됨.
> 구체적인 패시브 수치/그룹화는 별도 설계 필요 (→ 010 문서 예정).

### ★ 희귀 — 코돈 1개 (유일 경로)

| 아미노산 | 코돈 | 패시브명 | 효과 |
|---------|------|---------|------|
| 메티오닌 (Met) | ATG | **기폭** | 강력한 공격 패시브. 생물학의 "시작 코돈" |
| 트립토판 (Trp) | TGG | **변이촉발** | 변이 관련 강력한 패시브 |

### 일반 — 코돈 2개

| 아미노산 | 코돈 | 패시브명 | 효과 방향 |
|---------|------|---------|----------|
| 페닐알라닌 (Phe) | TTT, TTC | **폭발** | 공격력 증폭 |
| 타이로신 (Tyr) | TAT, TAC | **각인** | 디버프 지속 |
| 히스티딘 (His) | CAT, CAC | **상성폭발** | 상성 유리 시 추가 효과 |
| 글루타민 (Gln) | CAA, CAG | **반사** | 피해 반사 |
| 아스파라긴 (Asn) | AAT, AAC | **유전** | 유전/보존 관련 |
| 라이신 (Lys) | AAA, AAG | **선제** | 선공 보너스 |
| 아스파르트산 (Asp) | GAT, GAC | **반격** | 후공 보너스 |
| 글루탐산 (Glu) | GAA, GAG | **환경지배** | 환경 효과 강화 |
| 시스테인 (Cys) | TGT, TGC | **결합** | 인접 시너지 |

### 보통 — 코돈 3개

| 아미노산 | 코돈 | 패시브명 | 효과 방향 |
|---------|------|---------|----------|
| 이소류신 (Ile) | ATA, ATT, ATC | **침식** | 방어력 감소 누적 |

### 흔함 — 코돈 4개

| 아미노산 | 코돈 | 패시브명 | 효과 방향 |
|---------|------|---------|----------|
| 발린 (Val) | GTA, GTT, GTG, GTC | **흡수** | HP 흡수 |
| 트레오닌 (Thr) | ACA, ACT, ACG, ACC | **관통** | 방어 무시 |
| 알라닌 (Ala) | GCA, GCT, GCG, GCC | **경화** | 피해 감소 |
| 글리신 (Gly) | GGA, GGT, GGG, GGC | **경량화** | HP 소모 감소 |
| 프롤린 (Pro) | CCA, CCT, CCG, CCC | **강직** | 선공 보장 |

### 최흔 — 코돈 6개

| 아미노산 | 코돈 | 패시브명 | 효과 방향 |
|---------|------|---------|----------|
| 류신 (Leu) | TTA, TTG, CTA, CTT, CTG, CTC | **가속** | SPD 증가 |
| 세린 (Ser) | AGT, AGC, TCA, TCT, TCG, TCC | **재생** | HP 회복 |
| 아르기닌 (Arg) | AGA, AGG, CGA, CGT, CGG, CGC | **적응** | 상황 적응 |

### ■ 정지 코돈 — 위험

| 코돈 | 패시브명 | 효과 |
|------|---------|------|
| TAA, TAG, TGA | **유전자 침묵** | 이 코돈 구간 비활성화 (패시브 없음) |

## 희귀도-접근성 밸런스

```
경로 수 ↔ 효과 강도 (반비례)

★ 1경로 (Met, Trp)        = 유일한 조합. 빌드 정의급
   2경로 (Phe~Cys)         = 조건부 강력
   3경로 (Ile)              = 중간
   4경로 (Val~Pro)          = 안정적, 범용
   6경로 (Leu, Ser, Arg)    = 가장 쉬움, 기본 유틸
■ 3경로 (Stop)             = 함정. T로 시작 시 주의
```

## 효소(조커)와의 관계

코돈은 기본 패시브를 제공하고, 효소(조커)가 조건 매칭 시 **강력한 증폭/변형 효과**를 추가.

```
코돈 ATG(기폭) 기본 패시브: 데미지 +10%
  + [촉매 효소] 매칭: 기폭 코돈 → 데미지 +10% → +50%로 증폭!
```

효소 없이도 코돈은 항상 기본 패시브를 제공 → "꽝"인 코돈 없음.
