# 014. 체질 시스템 (Attribute System)

## 1. 핵심 개념

체질 = 실험체의 **선천적 스탯 파생 공식**을 결정하는 고정 특성.
같은 STR 30이라도 맹공 체질과 철벽 체질에서 HP와 데미지가 완전히 다름.

### 설계 원칙

- **6종 고정, 혼합 불가** — 체질은 뼈대. 단단하고 심플하게
- **공식 계수 변경 방식** — 비율 곱연산(후반 스노우볼)이나 플랫 가감(후반 무의미)이 아닌, 성장의 기울기(Growth Rate) 자체를 변경
- **유전 + 환경** — 부모로부터 후보 풀을 물려받되, 최종 결정은 연구실에서 유저가 "주입"

---

## 2. 기본 스탯 파생 공식 (균형 체질 = 기준)

### 4대 스탯

| 스탯 | 역할 | 범위 (예상) |
|------|------|-----------|
| **STR (힘)** | HP, 기본 데미지 보정 | 10~60 |
| **DEX (민첩)** | SPD(선후공), 회피 | 10~60 |
| **RES (내성)** | DEF(데미지 감소), 상태이상 저항 | 10~60 |
| **MUT (변이)** | 변이 빈도/품질/선택지 (013) | 0~100 |

### 파생 공식 (균형 체질 기준선)

```
HP         = 80 + (STR × 2.0)
기본 데미지 = 10 + (STR × 0.1)         ← 상성 승리 시 적용되는 기본값
SPD        = 10 + (DEX × 0.5)          ← 선후공 판정 기준
DEF        = RES × 0.3                 ← 피격 데미지 감소율 (%, 최대 30%)
변이 확률   = MUT / (MUT + 100)         ← 트리거당 변이 발동 확률
```

> SPD가 높은 쪽이 먼저 공격. SPD 동일 시 50:50.
> DEF는 % 감소: 데미지 10, DEF 15% → 실 데미지 8.5

## 3. 6종 체질 — 공식 계수 변경표

### 총괄표

| 체질 | HP 공식 | 기본 데미지 | SPD 공식 | DEF 공식 | MUT 분모 | 빌드 성격 |
|------|---------|-----------|---------|---------|---------|----------|
| **균형** | 80 + STR×**2.0** | 10 + STR×**0.10** | 10 + DEX×**0.5** | RES×**0.30** | **100** | 범용 기준선 |
| **맹공** | 80 + STR×**1.5** | 10 + STR×**0.18** | 10 + DEX×**0.5** | RES×**0.20** | **100** | 글래스캐논 |
| **철벽** | 80 + STR×**2.5** | 10 + STR×**0.05** | 10 + DEX×**0.4** | RES×**0.45** | **100** | 탱 |
| **신속** | 80 + STR×**1.5** | 10 + STR×**0.10** | 10 + DEX×**0.8** | RES×**0.30** | **100** | 선공 제압 |
| **재생** | 80 + STR×**2.5** | 10 + STR×**0.10** | 10 + DEX×**0.3** | RES×**0.30** | **100** | 장기전 |
| **변이** | 80 + STR×**2.0** | 10 + STR×**0.10** | 10 + DEX×**0.5** | RES×**0.20** | **70** | 카오스 |

> 볼드 수치가 균형 대비 변경된 계수. 각 체질은 2~3개 계수만 변경.
> 정확한 수치는 밸런싱 단계에서 조정 가능. 위 수치는 설계 기준선.

### 상세 설명

#### 맹공 (Aggression)

```
HP         = 80 + STR × 1.5    (균형 대비 HP 성장 -25%)
기본 데미지 = 10 + STR × 0.18   (균형 대비 데미지 성장 +80%)
DEF        = RES × 0.20         (균형 대비 방어 성장 -33%)
```

| STR | 균형 HP | 맹공 HP | 균형 데미지 | 맹공 데미지 |
|-----|--------|--------|-----------|-----------|
| 20 | 120 | 110 | 12.0 | 13.6 |
| 30 | 140 | 125 | 13.0 | 15.4 |
| 40 | 160 | 140 | 14.0 | 17.2 |
| 50 | 180 | 155 | 15.0 | 19.0 |

→ STR이 높을수록 격차 벌어짐. STR 50 맹공은 데미지 19로 기본 데미지의 거의 2배.
→ 대신 HP는 180 vs 155 = 25 차이. 탈진 2~3회면 치명적.

#### 철벽 (Fortress)

```
HP         = 80 + STR × 2.5    (HP 성장 +25%)
기본 데미지 = 10 + STR × 0.05   (데미지 성장 -50%)
SPD        = 10 + DEX × 0.4    (SPD 성장 -20%)
DEF        = RES × 0.45         (방어 성장 +50%)
```

→ HP 높고 DEF 높지만, 데미지가 처참하고 후공. 탈진전에 강함.
→ 시너지: 반응성 굳은살(연속 패배 감소) + 외골격(쉴드) + 탈진 싸움

#### 신속 (Agility)

```
HP         = 80 + STR × 1.5    (HP 성장 -25%)
SPD        = 10 + DEX × 0.8    (SPD 성장 +60%)
```

→ 항상 선공. 선공 = 상대가 반격하기 전에 파괴 = 피해 회피.
→ 시너지: 아드레날린 선(SPD 부스트) + 선독(신경독 가시)

#### 재생 (Regeneration)

```
HP         = 80 + STR × 2.5    (HP 성장 +25%)
SPD        = 10 + DEX × 0.3    (SPD 성장 -40%)
```

→ 높은 HP + 느린 속도 = 장기전 특화. 탈진 데미지를 HP로 버팀.
→ 시너지: 재생 세포(HP 회복/오버힐→쉴드) + 공생 미생물(동종 진화 회복) + 극저온 동결 캡슐

#### 변이 (Mutant)

```
DEF        = RES × 0.20         (방어 성장 -33%)
변이 확률   = MUT / (MUT + 70)   (균형 대비 분모 70, 변이 빈도 ↑)
```

| MUT | 균형 확률 | 변이 확률 |
|-----|---------|---------|
| 30 | 23% | 30% |
| 50 | 33% | 42% |
| 80 | 44% | 53% |
| 100 | 50% | 59% |

→ 변이 팝업이 훨씬 자주 뜸. MUT 80 변이 체질 = 매 트리거 절반 이상 발동.
→ 대신 DEF가 낮아서 맞으면 아픔 = 하이리스크 하이리턴.
→ 시너지: 돌연변이 안정제(선택지 추가) + 불안정 서열(플라스미드) + 키메라 노림

#### 균형 (Balanced)

```
모든 계수 기준선. 보정 없음.
```

→ 모든 스탯이 고르게 성장. 빌드 자유도 최고.
→ 초보자 추천 + "체질이 아닌 아미노산/효소로 특화하겠다"는 빌드에 적합.

## 4. 유전 방식 — 유전 + 환경 (Pool + Injection)

### 교배 시 체질 풀 생성

```
부모 A (맹공) + 부모 B (철벽) = 자식 체질 풀:
  [맹공]  — 부모 A에서 유전
  [철벽]  — 부모 B에서 유전
  [???]   — 돌연변이 슬롯 (MUT 스탯 기반)
```

### 풀 규칙

| 슬롯 | 출처 | 설명 |
|------|------|------|
| 슬롯 1 | 부모 A 체질 | 항상 포함 |
| 슬롯 2 | 부모 B 체질 | 항상 포함 (A=B면 슬롯 2도 같은 체질) |
| 슬롯 3 | 돌연변이 | 부모 양쪽에 없는 체질 중 랜덤 1개 |

- 부모 양쪽이 같은 체질(맹공+맹공)이면: 풀 = [맹공, 맹공, 랜덤] → 맹공 확정이 아닌, 유저가 여전히 선택
- 돌연변이 슬롯 등장 확률: **부모 중 높은 MUT / (MUT + 100)**
  - 부모 MUT 낮으면 돌연변이 슬롯 비활성 → 풀이 2개만
  - 부모 MUT 높으면 거의 항상 3번째 선택지 제공

### 연구실 "주입" (Injection)

```
┌─── 체질 결정 ───────────────────────────┐
│                                          │
│  🧬 부모 유전자 풀:                       │
│                                          │
│  [맹공]  — 부모 A (맹공)에서 유전         │
│  [철벽]  — 부모 B (철벽)에서 유전         │
│  [신속]  — 돌연변이 (MUT 50 발동!)        │
│                                          │
│  배양액을 선택하세요:                      │
│  ▶ 맹공 배양액 투입                       │
│    철벽 배양액 투입                        │
│    신속 배양액 투입                        │
│                                          │
└──────────────────────────────────────────┘

→ 유저가 "맹공" 선택
→ 자식 실험체: 체질 = 맹공 (확정, 영구)
```

### 비용 요소 (밸런싱 여지)

- 풀 내 체질 선택: 무료 (기본 유전)
- 돌연변이 슬롯 선택: 추가 비용? (연구 자원 소모)
- → 세부 비용은 경제 시스템 설계 시 확정. 현재는 "선택은 무료" 기본값.

## 5. 체질과 전투 시스템 시너지

### 체질별 추천 빌드 아키타입

| 체질 | 핵심 시너지 | 추천 아미노산 | 추천 효소 | 빌드 네이밍 |
|------|-----------|------------|---------|-----------|
| **맹공** | 높은 데미지 × 승리 보상 | 발화샘(크리), 탐식의 턱(킬 펌핑) | 피의 펌프 모터, A-스테로이드 앰플 | "버서커" |
| **철벽** | 높은 HP × 패배 보상 | 외골격(쉴드), 반응성 굳은살(연속 패배 감소) | 경화 수지 스프레이, 충격 흡수 겔 | "요새" |
| **신속** | 선공 × 선제 효과 | 아드레날린 선(SPD↑), 신경독 가시(선공 독) | 신경망 오버라이드 칩, G-시냅스 연결선 | "암살자" |
| **재생** | 높은 HP × 회복 | 재생 세포(회복), 공생 미생물(동종 회복) | 극저온 동결 캡슐, 세포 분열 억제제 | "좀비" |
| **변이** | 잦은 변이 × 키메라 | 박동하는 종양(변이촉발), 융합 점균(옵션 돌파) | 돌연변이 안정제, 유전자 재조합 칵테일 | "키메라 헌터" |
| **균형** | 범용 × 자유 조합 | 제한 없음 | 제한 없음 | "만능형" |

### 체질 × 탈진 전략

```
탈진 데미지 = 순환 횟수 × 5 (양쪽 동시)

철벽 (HP 180) vs 맹공 (HP 155):
  탈진 누적 50 시점: 철벽 HP 130 (생존), 맹공 HP 105 (위험)
  → 철벽은 탈진전에서 유리

맹공은 탈진 전에 끝내야 함:
  → 높은 데미지로 적 Gene 전소 or HP 0 달성
  → 죽음의 회전목마(후반 순환)에 약함
```

## 6. 체질 변경 불가 — 영구 고정

- 체질은 탄생 시 확정, 이후 **변경 불가**
- 플라스미드, 효소, 아이템으로도 체질 변경 불가
- 체질이 마음에 안 들면 → 교배로 새 실험체 생산 (세대 교체)
- → 체질 = 실험체의 "종(Species)" 수준의 근본 특성
