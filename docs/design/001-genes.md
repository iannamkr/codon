# Gene (유전자) 설계

## 정의

Gene = **A, T, G, C** 4종류의 염기. 코돈(스킬)을 만들기 위한 구성 재료.

```
Gene은 스킬이 아니다.
Gene 3개를 조합한 코돈이 스킬이다.

[A] [T] [G] → 코돈 ATG → 스킬 "기폭"
```

## Gene 목록

| Gene | 이름 |
|------|------|
| **A** | 아데닌 (Adenine) |
| **T** | 티민 (Thymine) |
| **G** | 구아닌 (Guanine) |
| **C** | 시토신 (Cytosine) |

4개 고정. 추가/확장 없음.
