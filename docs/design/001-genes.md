# Gene (유전자) 설계

## 정의

Gene = **A, T, G, C** 4종류의 염기. 전장에 배치되어 1:1로 전투하는 **물리적 블록**.

```
Gene은 하위 효과가 내장된 완성품 아이템이다.
[강타하는 A], [은신하는 C] 등 — 드래그 앤 드롭 1회로 배치.

코돈 판정: 베이스(A/T/G/C)만 읽음 → 패턴 매칭
전투 판정: 상성(A>T) + 하위 효과(강타 +15%) 개별 적용
```

## Gene 목록

| Gene | 이름 | 역할 | 이기는 상대 | 지는 상대 |
|------|------|------|-----------|-----------|
| **A** | 아데닌 (Adenine) | 파괴 (Destroy) | T | C |
| **T** | 티민 (Thymine) | 생존 (Survive) | G | A |
| **G** | 구아닌 (Guanine) | 질서 (Order) | C | T |
| **C** | 시토신 (Cytosine) | 혼돈 (Chaos) | A | G |

4개 고정. 추가/확장 없음.

## 순환 상성

```
A(파괴) > T(생존) > G(질서) > C(혼돈) > A(파괴)
```

- 파괴 > 생존: 파괴가 생존을 부순다
- 생존 > 질서: 압도적 생명력이 체계를 무시한다
- 질서 > 혼돈: 체계가 혼란을 억제한다
- 혼돈 > 파괴: 예측불가능이 파괴를 무력화한다

## 일체형 아이템 모델 (Integrated Gene)

각 Gene(A/T/G/C)에 하위 Gene 10종이 존재 (총 40종).
인벤토리의 유전자 블록 자체가 하위 효과를 내장한 완성품.

```
[강타하는 A] = 베이스 A + 하위 효과 "강타"
  → 코돈 판정 시: A로 읽힘
  → 전투 판정 시: 강타 보너스(위력 +15%) 적용
```

파밍 동기:
- 초보: A면 아무거나 대충 넣어서 코돈 시너지 맞춤
- 고인물: "3번 자리에 [흡혈하는 A]가 뜰 때까지 합성/배양 노가다"
