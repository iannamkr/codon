# 논의 003: Attribute (체질)

## 발단

스탯(힘/민첩/내성/변이) 확정 후 Attribute 논의로 넘어감.

## Stat vs Attribute 구분

- Stat = 전투 판정용 기본 수치 (힘/민첩/내성/변이)
- Attribute = 실험체의 고유 특성

## Mewgenics 참고

사용자:
> "mewgenics를 참고해볼까?"

Mewgenics 분석 결과:

**전투 스탯 7개:** STR, DEX, CON, INT, SPD, CHA, Luck

**성격/행동 특성 (Attribute):**
- Libido (높/보통/낮) — 교배 빈도
- Aggression (높/보통/낮) — 싸움 빈도
- 성적 지향 — 교배 대상

**클래스 + 어빌리티:**
- 13개 클래스, 클래스당 75개 어빌리티 (총 1200+)
- 패시브 최대 2개 + Disorder 최대 2개

**핵심 구조:**
```
Mewgenics 고양이 = 전투 스탯 + 성격 특성 + 클래스 + 어빌리티 + Disorder + 변이
```

## MUT를 스탯으로 승격

사용자:
> "우리도 기본 스탯에 Luck대신 MUT를 넣을까?"

→ MUT를 4번째 기본 스탯으로 승격. Mewgenics의 Luck 역할(크리티컬, 랜덤)을 MUT가 흡수.

확정:
```
힘 (STR) → HP, 데미지
민첩 (DEX) → SPD, 선공
내성 (RES) → DEF, 속성 저항
변이 (MUT) → 전투 중 변이 횟수, 크리티컬, 변이 품질
```

## Attribute 방향 논의

MUT가 스탯으로 올라가면서, Attribute는 전투 외적인 것만 남는다는 전제로 시작했으나...

사용자:
> "빌드에 영향을 끼치는 attribute 면 좋겠어."

→ 방향 전환. Attribute는 전투 외적이 아니라 빌드에 영향을 주는 것으로.

### Attribute가 빌드에 영향 주는 방식

스탯과의 차이:
```
스탯 = 숫자. 얼마나 강한지 (scaling)
Attribute = 성질. 어떻게 작동하는지 (mechanics 변경)
```

PoE 비유: 스탯 = STR/DEX/INT, Attribute = 어센던시/키스톤

### 후보 방향 3개

**A. 키스톤형 — 빌드 규칙을 바꾸는 특성**
```
"과발현" — 코돈 시너지 효과 ×2, 대신 정지 코돈 발동 확률 +30%
"냉혈" — 내성 파생이 2배, 대신 힘 파생 절반
"불안정 서열" — 슬롯 1개 추가, 대신 매 전투 코돈 1개가 랜덤 비활성화
```
- 장점: 같은 코돈/스탯이라도 완전히 다른 빌드. 빌드 다양성 폭발
- 단점: 설계/밸런싱 난이도 높음

**B. 속성 친화형 — 특정 코돈 계열을 강화**
```
"Fire 친화" — Fire 속성 코돈 효과 +30%
"A계열 특화" — A로 시작하는 코돈 데미지 +25%
```
- 장점: 직관적. 교배 시 방향성
- 단점: 단순. 스탯과 차이 모호

**C. 생물학적 체질 — 파생 공식 자체를 변형**
```
"재생체질" — 힘에서 HP 파생 비율 1.5배, 대신 데미지 파생 0.7배
"돌연변이체" — MUT에서 변이 품질 파생이 2배, 대신 안정도 절반
"경량체" — 민첩 파생 전부 1.3배, 대신 HP 총량 -20%
```
- 장점: 같은 스탯이라도 체질에 따라 다른 빌드. 교배 시 체질 유전이 전략
- 단점: 파생 공식 복잡해질 수 있음

### A+C 제안 vs 복잡도 우려

제안: A+C 혼합 (체질로 파생 분화 + 키스톤으로 규칙 변경)

사용자:
> "a, c로 가면 너무 게임이 어렵지 않을까? 아니 이미 돌이킬 수 없는 강을 건넜나? ㅋㅋㅋㅋㅋㅋㅋㅋㅋ"

이미 확정된 시스템이 많아서 A+C는 과함. 가벼운 대안으로 B(속성 친화)만 제안했으나:

사용자:
> "난 속성 친화로 가면 재미 없을 것 같은데? 너무 큰 틀에서 강제가 되잖아? 그리고 상성도 너무 고정되고. C가 맞지 않아?"

→ B는 "이 실험체 = Fire 타입"이 되어서 자유도 감소. C(체질)이 맞다.

### 체질 6종 확정

키스톤(A)은 발현 트리 최종 노드에서 제공하기로 하고, 체질(C)만 Attribute로:

| 체질 | 버프 | 너프 | 빌드 성격 |
|------|------|------|----------|
| **맹공** | 데미지 파생 ↑ | DEF 파생 ↓ | 글래스캐논 |
| **철벽** | DEF + 속성저항 ↑ | 데미지 파생 ↓ | 탱 |
| **신속** | SPD + 선공 ↑ | HP 파생 ↓ | 선공 제압 |
| **재생** | HP 파생 ↑ | SPD 파생 ↓ | 장기전 |
| **변이** | 변이횟수 + 크리티컬 ↑ | 내성 파생 전체 ↓ | 하이리스크 하이리턴 |
| **균형** | 보정 없음 | 보정 없음 | 범용 |

### 결정

- **체질 6종 방향 확정**
- 세부 수치, 유전 방식, 혼합 체질 여부는 **추후 재논의**
- 키스톤급 효과는 체질이 아닌 **발현 트리 최종 노드**에서 제공
