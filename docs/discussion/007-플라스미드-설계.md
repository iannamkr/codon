# 논의 007: 플라스미드 설계

## 배경

하위 Gene 40종 + 속성 시스템 확정 후, 플라스미드(Layer 2) 세부 설계로 넘어감.
기존 확정: 코돈 = 순수 효과, 플라스미드 = 별도 레이어로 속성/변형 부여 (006에서 결정).

## 플라스미드의 역할: 강화 vs 예외

사용자:
> "강화할지 예외성을 부여할지 판단이 어려워. 다키스트 던전의 스트레스 시스템이나 mewgenics, reference poe 문서에서 키스톤의 사례를 보여줄래?"

### 레퍼런스 조사 결과

**PoE 키스톤** (15종 분석):
- Chaos Inoculation: Chaos 면역, 대신 최대 HP = 1
- Avatar of Fire: 모든 데미지 → Fire 전환. 비Fire 불가
- Blood Magic: 마나 삭제, 스킬을 HP로 시전
- Ancestral Bond: 직접 데미지 불가, 토템 +1
- Iron Reflexes: 회피 → 방어로 전환
- Resolute Technique: 크리 불가, 대신 100% 명중
- Mind Over Matter: 피해 30%를 마나로 흡수
- Pain Attunement: 로우 라이프 시 데미지 30%↑
- → 공통 패턴: 전부 "A를 포기하는 대신 B를 얻는다". 숫자가 아니라 규칙이 바뀜

**Darkest Dungeon** (스트레스 시스템):
- 스트레스 100 → 고난(Affliction, 75%) 또는 미덕(Virtue, 25%)
- 고난: 비겁(도주), 편집증(힐 거부), 이기적(아이템 독점), 학대적(동료 공격) — 컨트롤 상실
- 미덕: 용맹(ATK↑), 집중(크리↑) — 초과 보상
- → 패턴: 같은 시스템이 제어 불능 또는 초과 보상. 리스크 판단

**Mewgenics** (변이):
- Wings: 벽 무시 이동
- No Mouth: 먹기 불가, 대신 광합성 회복
- Glass Body: HP 1, 회피 극대화
- Extra Limbs: 행동 횟수 +1
- → 패턴: PoE와 동일하게 "규칙 자체를 변경"

### 3개 레퍼런스 공통 결론

"X를 못 하는 대신 Y를 얻는다" — 규칙 파괴가 빌드 정체성을 만든다.

## 결정: 규칙 파괴형

사용자:
> "규칙 자체를 바꾸거나 파괴하는 방식이 재미있을 것 같아."

확정: 플라스미드 = **규칙 파괴형** (PoE 키스톤 패턴). 숫자 보너스가 아닌 규칙 변경.

## 빌드 순서 문제

제시한 12종 후보 중 10번(과부하: 코돈 3→2개/시퀀스, 4→5페이즈)처럼 빌드 구조 자체를 변경하는 플라스미드가 있으면, 시퀀스를 짜기 전에 플라스미드를 먼저 골라야 하는 문제 발생.

사용자:
> "만약 10번 같은 기능을 제공하려면 항상 플라스미드를 먼저 배치해야겠네? 빌드 짤때?"

### 방안 A: 플라스미드를 맨 먼저 선택
```
플라스미드 선택 → (규칙 변경 적용) → 코돈/시퀀스/빌드 구성
```
PoE에서 키스톤 먼저 찍고 빌드 맞추는 것과 동일. 다양성 극대화.

### 방안 B: 플라스미드는 전투 규칙만 변경 (빌드 구조 안 건드림)
구조 변경형 플라스미드 제외. 범위가 좁아짐.

## 결정: 방안 A

사용자:
> "A로 가는게 다양성에서 더 재미있을 것 같아"

확정:
- 플라스미드 = 빌드의 첫 번째 선택 (규칙 프레임)
- 빌드 순서: 플라스미드 → 코돈 → 시퀀스 → 빌드 배열
- 플라스미드 변경 시 빌드 전체 재구성 필요

## 12종 후보

사용자:
> "응 밸런스는 아직 못맞추겠어 진행하고 나중에 수정하자."

12종 후보 잠정 확정 (수치 밸런싱 미정):
1. 역행 (전투: SPD 제거 → 역순 발동)
2. 공명체 (전투: 인접 상호작용 제거 → 같은 태그 3개면 ×3)
3. 거울 (전투: 내 배열 무효 → 상대 복사)
4. 선독 (전투: Phase 1~3 제거 → Phase 4만 ×4)
5. 불안정 서열 (변이: 선택 제거 → 자동 수락 + 확률 2배)
6. 순수 서열 (변이: 변이 제거 → 스탯 +20%)
7. 적응체 (변이: 영구 제거 → 전투 중만, 효과 ×0.5)
8. 무속성 (속성: 상성 제거 → ×1.0 + 순수 뎀 +30%)
9. 키메라 (속성: 단일 속성 제거 → 랜덤 속성 + 유리 시 ×1.5)
10. 과부하 (구조: 코돈 3→2, 페이즈 4→5)
11. 압축 (구조: 4페이즈 → 2페이즈, 코돈 6개 충돌)
12. 기생체 (메타: 상대 플라스미드 무효화 + 탈취)
