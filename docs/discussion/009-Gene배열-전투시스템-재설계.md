# 009. Gene 배열 전투 시스템 재설계

## 발단 — 유저 제안

유저가 새로운 전투 시스템 아이디어를 제안:

> - Gene을 12개 배치하고 전투시작하면 배치한 gene의 순서대로 3개씩 코돈으로 판단해서 효과를 발동
> - 전투시 gene이 1:1로 부딪히며 gene끼리는 상성이 있어서 우열한 쪽의 gene이 사라지고 다음 오른쪽 gene이 당겨지며 다음 턴에 다시 코돈 효과 새롭게 발동
> - 체력이 0이 되거나 gene이 모두 소모되면 패배, 퍼즐 요소를 가미
>
> 문제:
> - Gene을 12개 배치하는 과정을 유저가 어떻게 인식하게 할 것인가?
> - 조합과 빌드 컨셉을 어떻게 유도할 수 있는가?

## 분석 — C안 채택

3가지 후보 방향 제시:
- A안: 현재 시스템 유지 + 퍼즐 요소 부분 도입
- B안: Gene 12배열 전면 전환 (기존 설계 전부 폐기)
- C안: 하이브리드 — 빌드 시점에는 코돈 단위로 사고, 전투는 Gene 단위로 수행

**유저 결정: C안 채택** + 발라트로의 족보/조커 개념 도입

> 빌드 시점부터 Gene 단위로 하면 유저가 빌드의 컨셉을 정하기 어렵고, Gene의 상성에 따라 어차피 운빨이라는 생각이 강함. 이를 해결하기 위하여 발라트로의 족보 개념 도입, 코돈이 마치 발라트로의 조커 카드 개념으로 글로벌 영향을 주는 개념.

## 4대 핵심 결정

### 1. 족보 인식 범위: 고정 윈도우

Gene 12개를 항상 3개씩 끊어 읽음 (고정 윈도우). 슬라이딩 윈도우 아님.

```
[A][T][G] [C][A][A] [T][G][C] [A][T][G]
 코돈1     코돈2     코돈3     코돈4
```

프레임시프트 = Gene이 빠지면 서열이 당겨져서 고정 윈도우가 읽는 내용이 바뀌는 것. 윈도우 규칙(3개씩) 자체는 변하지 않음. 실제 생물학의 읽기 틀 이동(frameshift mutation)과 동일.

### 2. 코돈(조커) 발동 방식: 코돈별 다양한 타입

매칭 발동 / 상시 패시브 / 하이브리드 — 코돈에 따라 다양한 효과 타입 혼합.

> 유저: "A, B, C 모두 포함, 코돈에 따라서 다양한 타입의 효과가 있어야 재미있을듯"

코돈(조커) 4개 장착. 연구실에서 세팅, 전투 중 불변. 인벤토리 개념.

### 3. Gene 상성: 순환 상성 (A>T>G>C>A)

기존 2축 대립(A-T, G-C)에서 4순환 상성으로 변경.

> 유저: "상성은 순환으로 변경, 필요하면 파괴/생존/질서/혼돈 다시 논의"

### 4. 전투 구조: 12 Gene = 1전투

4페이즈 구조가 아닌 12 Gene 전체가 하나의 연속 전투.

## Gene 전투 메커니즘 — 동종 돌파 / 이종 교대

### 기본 규칙

1. Gene이 1:1로 싸움 (앞에서부터 순서대로)
2. 상성 우위 쪽이 승리 → 패배한 Gene 소멸
3. 승리한 Gene은 전장에 생존
4. 다음 내 Gene이 동종이면 → **돌파** (합체하여 진화, A+A=A⁺)
5. 다음 내 Gene이 이종이면 → **교대** (생존 Gene이 Survivor Pool로 퇴장, 다음 Gene 출전)
6. 동등(같은 Gene)이면 → 양쪽 보존, 다음 Gene으로 넘어감 (데미지 없음)

### 유저 원문

> A가 승리하면 A는 보존되는거야. 그래야 승리한 경우 재출전 할때 Gene이 그대로 있지. 이겨도 소멸되면 의미가 없어.
>
> 제안: "동종은 진화(돌파), 이종은 임무 교대(보존)"
> 이종 결합(A+G)이라는 억지 개념을 아예 없애버리는 방향입니다.

### 전투 시뮬레이션 예시

```
[A₁][A₂][G₃][T₄] vs [T₁][C₂][A₃][G₄]

R1: A₁ vs T₁ → A₁ 승리 (A>T), A₁ 생존
    다음 내 Gene: A₂ (동종) → 돌파!

R2: A₁+A₂ = A⁺ vs C₂ → A⁺ 승리, A⁺ 생존
    다음 내 Gene: G₃ (이종) → 교대
    A⁺ → Survivor Pool. G₃ 출전

R3: G₃ vs A₃ → G₃ 패배 (A>G), G₃ 소멸
    적 A₃ 생존, 다음 적 Gene: G₄ (이종) → 교대
    적 A₃ → 적 Survivor Pool. 적 G₄ 출전

R4: T₄ vs 적 G₄ → T₄ 승리 (T>G... 상성 확인 필요)
    내 Gene 소진 → 전투 종료

결과:
내 Survivor Pool: [A⁺], [T₄]
내 소멸: [G₃]
```

## 족보와 코돈 효과 — 2레이어 구조

### 족보 vs 코돈 효과 혼동 정리

발라트로 대응:

| 발라트로 | 우리 게임 | 성격 |
|---------|----------|------|
| 카드 | Gene (A/T/G/C) | 기본 재료 |
| 족보 (페어, 플러시) | 빌드 시 확정된 코돈 4개 | 전투 내내 고정 패시브 |
| 조커 카드 | 장착한 코돈 4개 | 족보/코돈효과를 증폭하는 글로벌 효과 |
| 플레이한 핸드 점수 | 라운드별 코돈 효과 | 실시간 변동 |

### 2레이어 구조 확정

```
레이어 1: 족보 (빌드 시 확정, 전투 내내 유지)
───────────────────────────────────
빌드 시점에 12 Gene 배열 → 고정 윈도우로 4코돈 읽음
이 4코돈이 "족보" = 전투 내내 유지되는 글로벌 패시브 효과
전투 중 프레임시프트가 일어나도 족보는 변하지 않음

레이어 2: 코돈 효과 (전투 중 실시간 변동)
───────────────────────────────────
전투 중 Gene이 소멸되면 서열 당김(프레임시프트) 발생
남은 Gene을 다시 3개씩 읽음 → 새로운 코돈 효과 발현
매 라운드 변동 가능 → 전투의 동적 요소
```

### 구체 예시

```
═══ 빌드 시점 ═══
[A][T][G] [C][A][A] [T][G][C] [A][T][G]
 ATG       CAA       TGC       ATG
 기폭      글루타민    시스테인   기폭
→ 족보 확정: "기폭/글루타민/시스테인/기폭" = 전투 내내 고정

═══ 전투 중 (A 소멸 → 당김) ═══
[T][G][C] [A][A][T] [G][C][A] [T][G]
 TGC       AAT       GCA       TG(불완전)
 시스테인   아스파라긴   알라닌    ← 실시간 코돈 효과

→ 족보는 여전히 "기폭/글루타민/시스테인/기폭" (불변)
→ 코돈 효과만 변동
```

## 실험체 = 서열 영속 아이디어

전투 후 결과가 실험체에 영구 반영되는 개념:

> 유저: 하나의 시퀀스는 하나의 실험체에 매핑되어있음. 연구실에는 여러마리의 실험체가 존재 (빌드 갯수). 특정 실험체를 선택해서 전투 실행. 전투 후 승리해서 살아남은 경우 소멸된 gene은 제외하고 보존된 gene 또는 동종결합/버프부여된 gene이 유지됨. 연구실에서 해당 gene 뒤에 다시 12개의 gene을 채워서 실험체 조작 가능. 단 전투에 한번이라도 참가하면 이미 사용/결합된 gene은 수정 불가. 추가만 가능.

이 개념은 기존 설계의 열화/세대교체/유전자뱅크를 자연스럽게 대체:
- 열화 → Gene 잠김 누적 (전투할수록 자유도 감소 = 자연 열화)
- 세대 교체 → 잠긴 Gene이 대부분 → 은퇴/교배
- 유전자 뱅크 → Survivor Pool의 Gene을 보관
- 변이 → 전투 중 진화(A⁺)가 영구 잠김 = 의도치 않은 변이

## 후속 결정 — 미결 사항 논의

### 결정 1: 코돈(조커) 재평가 타이밍 → B안 (3라운드 단위)

**후보:**
- A안: 매 라운드 재평가
- B안: 3라운드 단위 재평가
- C안: 이벤트 트리거 재평가

**유저 결정: B안** — 3라운드 단위로 코돈 효과 재평가.

> Gene이 소멸되어 프레임시프트가 발생해도, 코돈 효과는 즉시 바뀌지 않음. 3라운드마다 남은 Gene을 다시 3개씩 읽어 새 코돈 효과 발현. 전투의 리듬감 제공 + 플레이어가 프레임시프트 효과를 예측할 수 있는 여유 부여.

### 결정 2: 전투 종료 조건 → C안 (Gene 전소 OR HP 0)

**후보:**
- A안: Gene 전소만
- B안: HP 0만
- C안: Gene 전소 OR HP 0

**유저 결정: C안** — Gene이 모두 소멸되거나 HP가 0이 되면 패배.

> 두 가지 패배 조건이 공존하여 전략 다양성 확보. Gene 수를 아끼는 생존형 빌드 vs HP를 깎는 공격형 빌드.

### 결정 3: 빈 자리 추가 규칙 → A안 진화: CRISPR/Splicing 메타포

**후보:**
- A안: 자유 배치 (생존자 사이에 삽입 가능)
- B안: 뒤에만 추가

**유저 결정: A안 진화** — 생존자가 압축 귀환, 잠긴 생존자 사이에 새 Gene 삽입 (Splicing).

> 전투 후 생존한 Gene은 **잠긴 상태로 압축 귀환**. 소멸된 Gene 자리가 빈 칸으로 남아, 유저가 새 Gene을 그 사이에 끼워넣음. 마치 DNA 수선(CRISPR) 과정처럼 — 손상된 서열을 잘라내고 새 서열을 이어붙이는 것.
>
> **핵심 규칙:**
> - 잠긴 Gene: 위치 이동/제거/교체 불가
> - 빈 칸: 새 Gene 자유 삽입 (드래그 → 스냅)
> - 프레임시프트와 연구실이 동일한 규칙: 전투 중 당김 = 연구실에서 삽입 후 읽기
> - 전투를 거듭할수록 잠긴 Gene 비율 증가 = 자연 열화 (자유도 감소)
> - 잠김이 대부분이면 은퇴/교배 시점 = 세대 교체

### 결정 4: 순환 상성 세부 → A안 (역할 유지, 상성만 순환)

**후보:**
- A안: 역할명(파괴/생존/질서/혼돈) 유지, 상성 관계만 순환으로
- B안: 역할명을 순환에 맞게 완전 재정의
- C안: 2축(빌드 분류)과 순환(전투 판정) 이중 체계

**유저 결정: A안** — 파괴/생존/질서/혼돈 그대로, 상성만 순환.

```
A(파괴) > T(생존) > G(질서) > C(혼돈) > A(파괴)
```

서사 해석:
- 파괴 > 생존: 파괴가 생존을 부순다
- 생존 > 질서: 압도적 생명력이 규칙/체계를 무시한다
- 질서 > 혼돈: 체계가 혼란을 억제한다
- 혼돈 > 파괴: 예측불가능이 파괴를 무력화한다

기존 하위 Gene 40종, 코돈 역할 태그 변경 없음.

### 결정 5: 코돈(조커) 전투 후 잠김 여부 → A안 (잠기지 않음)

**후보:**
- A안: 코돈(조커) 잠기지 않음 — 매 전투 전 자유 교체
- B안: 코돈(조커)도 전투 후 영구 잠김
- C안: 전투마다 1개씩 순차 잠김

**유저 결정: A안** — 코돈(조커)은 잠기지 않음. 자유 교체.

> Gene 배열이 전투마다 잠기면서 프레임시프트로 코돈 효과(레이어 2)가 변하는데, 그 변화에 맞춰 코돈(조커)을 재조정하는 것이 연구실 귀환 후 빌드 조정의 핵심 루프. 열화는 Gene 잠김만으로 충분.

### 패러다임 전환 — 코돈 = 패턴 (컨테이너 아님)

유저의 핵심 인사이트:

> 코돈의 개념은 이제 유전자를 담는 '물리적인 상자(Container)'가 아니라, '패턴(족보)'이자 '효과'라는 무형의 개념으로 완전히 전환된 것.

**과거 구조:** Gene → Codon(컨테이너) → Sequence(컨테이너) → Build = "폴더 속의 폴더"
**현재 구조:** 12개 Gene 서열(물리) + 코돈 패턴(무형) + 조커(효소/디코더)

컨테이너 계층이 사라지고, **Gene 서열이 유일한 물리적 실체**. 코돈은 서열을 읽는 "렌즈", 조커는 그 렌즈에 반응하는 "트리거".

### 용어 재정의

| 용어 | 정의 | 성격 |
|------|------|------|
| **Gene** (유전자/염기) | 전장에 배치, 1:1 전투, 소멸/보존되는 물리적 블록 (A/T/G/C) | 물리 |
| **DNA Chain** (서열) | 12개 Gene이 이어진 하나의 전장/탄창 | 물리 |
| **Codon** (코돈/족보) | 3개 Gene이 만드는 패턴 (예: "1구역 = A-A-T") | 무형 |
| **조커** (효소/디코더) | 연구실에서 장착하는 패턴 인식 트리거 (예: "A-A-T 읽으면 공격력 2배") | 장비 |

> 조커의 게임 내 명칭 후보: **효소(Enzyme)** — 생물학적으로 "특정 기질(패턴) 인식 → 반응 촉발"이 실제 효소의 역할과 동일. 세계관 일관성 높음. 최종 네이밍은 추후 확정.

### 결정 6: 실험체당 Gene 총량 한도 → A+안 (12칸 고정, 결정화)

**후보:**
- A안: 항상 12개 고정, 소멸 자리 채워서 복원
- B안: 누적 성장 — 전투마다 서열 길어짐
- C안: 상한 있는 누적 (최대 N개)
- A+안 (유저 제안): 12칸 고정 + 잠김 밀도가 성장/노화 척도

**유저 결정: A+안** — 12칸 결정화(Crystallization)와 노화(Aging).

> Gene 총량은 항상 12개로 고정. 잠긴 Gene이 쌓이는 것 자체가 성장과 노화의 척도. 길이가 아니라 **밀도**가 성장.

**실험체 생애 주기:**

```
탄생 (1전투):  [○][○][○][○][○][○][○][○][○][○][○][○]  자유 12칸. 유연성 100%
성장 (2~3전투): [★][★][★][★][○][○][○][○][○][○][○][○]  잠김 4 + 자유 8. 퍼즐 재미 최고조
노화 (4~5전투): [★][★][★][★][★][★][★][★][★][★][○][○]  잠김 10 + 자유 2. 깡스탯↑ 유연성↓
결정화 (최종):  [★][★][★][★][★][★][★][★][★][★][★][★]  잠김 12. 더 이상 빈칸 없음 = 은퇴
```

**핵심 딜레마:** 결정화된 실험체는 깡스탯은 역대급이지만 코돈 시너지 조정 불가.
- 극악 보스전에 투입 (명예로운 죽음)
- 연구실에서 은퇴시켜 메타 재화(플라스미드 등) 추출

**설계 장점:**
- 전투 스케일 항상 12 vs 12 = 밸런싱 일관성
- UI 고정 = 직관적
- 열화/세대교체가 별도 시스템 없이 자연 발생
- "강하지만 굳은 베테랑 vs 약하지만 유연한 신생" 딜레마

### 결정 7: 하위 Gene의 위치 → A+안 (일체형 아이템 모델)

**후보:**
- A안: Gene 블록에 하위 Gene 직접 부착 (2단계 배치)
- B안: 코돈 패턴 매칭 시 자동 결정 (조커에 내장)
- C안: 하위 Gene 제거, 조커에 통합
- A+안 (유저 제안): 일체형 아이템 모델 — Gene 자체가 하위 Gene을 포함한 완성품

**유저 결정: A+안** — Integrated Gene 모델.

> 인벤토리의 유전자 블록 자체가 `[강타하는 A]`, `[은신하는 C]` 같은 하나의 완성된 아이템. 배치 시 드래그 앤 드롭 1회로 끝.

**시스템 작동:**
1. **배치**: 인벤토리에서 `[강타하는 A]`를 12칸 중 하나에 드롭. 1회 조작
2. **코돈 판정**: 베이스(A/T/G/C)만 읽음. 하위 효과 무시. "A-T-G → 기폭 조커 발동!"
3. **전투 판정**: 상성(A>T)으로 승패 + 하위 효과(강타 +15%) 개별 적용

**파밍/크래프팅 동기:**
- 초보: A면 아무거나 대충 넣어서 코돈 시너지 맞춤 (일단 굴러감)
- 고인물: "3번 자리에 [흡혈하는 A]가 올 때까지 합성/배양 노가다" (PoE식 옵션 타협 거부)

**결정화 시너지:**
- 잠긴 `[강타하는 A]` = "5번 진화한 전설의 강타 유전자" → 서사 + 애착
- 디아블로/PoE의 접두사/접미사 모델을 생물학 테마로 번역

### 결정 8: 족보와 코돈(조커)의 효과 프레임워크

#### 족보(코돈) = 아미노산 기본 패시브 (Innate Effect)

**결정:** 족보 자체가 기본 효과를 제공 + 조커가 증폭.

> 코돈이 단순 조건이면 64개 중 조커에 해당하지 않는 60개가 "꽝"이 되어 프레임시프트 시 극심한 불쾌감 발생. 따라서 모든 코돈이 기본 패시브(아미노산)를 제공해야 함.

```
코돈(족보) = 아미노산 (기본 패시브)
────────────────────────────────
3개의 Gene이 코돈으로 읽히면, 조커 유무와 상관없이
해당 페이즈(3라운드) 동안 기본 패시브 제공.

예: [A-T-G] → '기폭 아미노산' → "이번 3라운드 데미지 +10%"

개발 효율: 64개 코돈을 아미노산 그룹으로 묶음 (생물학 중복성 활용)
A 다수 = 공격형, T 다수 = 방어형, G 다수 = 제어형, C 다수 = 교란형
```

```
조커 = 효소 (규칙 파괴자 / 증폭기)
────────────────────────────────
기본 코돈이 형성되는 것을 지켜보다가,
조건이 맞으면 개입하여 판을 뒤집는 강력한 효과 부여.
```

#### 조커(효소) 트리거 = 3가지 카테고리 (혼합형)

**결정:** 패턴 인식 / 조건 인식 / 이벤트 인식 혼합.

**1. 패턴 인식형 (족보 매칭)**
- 특정 코돈(아미노산)이 완성되었을 때 시너지
- 예: `[촉매 효소]` — "기폭(ATG) 코돈 형성 시, 기본 +10% → +50%로 증폭"
- 유저 심리: "ATG가 깨지지 않게 배열을 맞춰야지!"

**2. 조건 인식형 (Gene/상태 매칭)**
- 코돈 모양과 무관, 윈도우 내 특정 염기나 상황에 반응
- 예: `[A-편애 효소]` — "3칸 안에 A 존재할 때마다 체력 5 회복" (A-A-A면 15)
- 유저 심리: "족보 포기, A를 무식하게 깔아서 좀비처럼 버텨야겠다!"

**3. 이벤트 인식형 (프레임시프트/전투 반응)**
- 1:1 전투 결과나 프레임시프트 자체에 반응
- 예: `[포식자 효소]` — "내 Gene이 적 Gene을 파괴하고 생존할 때마다, 다음 1회 공격력 2배"
- 유저 심리: "초반에 상성 우위 Gene을 몰아넣어 스노우볼!"

#### 1페이즈(3라운드) 작동 시나리오

```
1. [스캔]     시스템이 1~3번 슬롯 [A-A-T] 스캔
2. [아미노산]  "A 2개 이상 → <근력 아미노산> 생성! 공격력 +10%"
3. [조커A]    패턴 인식: "A-A-T 족보? 적 전체 출혈 부여!"
4. [조커B]    조건 인식: "윈도우에 T 존재? 쉴드 +10!"
5. [전투]     3턴 1:1 박터짐. 이벤트형 조커C: 킬 낼 때마다 힐
6. [시프트]    소멸/생존 → 프레임시프트 → 새 윈도우 스캔 대기
```

### 결정 9: 플라스미드 재설계 방향 → C안 (보류 후 역설계)

**후보:**
- A안: 12종 전면 재설계
- B안: 유효한 것 살리고 무효한 것만 교체
- C안: 코돈/조커 효과 확립 후 역설계

**유저 결정: C안** — 플라스미드 논의 보류. 기본 규칙 확립 후 역설계.

> "기본 규칙(Standard Rule)이 100% 견고하게 세워지기 전에, 규칙 파괴자(Rule Breaker)를 먼저 만들면 게임이 산으로 간다."

**이유:**
1. 플라스미드 = 규칙 파괴. 기준점(코돈 효과, 하위 Gene 수치)이 없으면 파괴할 대상이 없음
2. 코돈/조커 효과 리스트업 → 유저 선호 콤보 파악 → 그 콤보를 비트는 플라스미드 역설계 = 버려지는 플라스미드 0
3. 새 코어 룰(프레임시프트, 결정화, 윈도우 스캔)이 엄청난 비틀기 잠재력 보유

**역설계 시 나올 법한 방향 예시 (미리보기):**
- `[역행자]`: 당겨짐이 앞→뒤가 아니라 뒤→앞 (배치 퍼즐 역전)
- `[과성장]`: 코돈을 3칸이 아니라 4칸(Tetrad) 단위로 읽음 (발동 어렵지만 효과 3배)
- `[재조합]`: 잠긴 Gene을 새 Gene으로 덮어쓸 수 있음 (대가: 최대 체력 영구 감소)

**기존 12종 플라스미드 상태:**
- 완전 무효 (3종): 공명체, 선독, 압축 (의존하던 시스템 폐기)
- 재정의 필요 (4종): 역행, 거울, 과부하, 키메라
- 변이 시스템 의존 — 보류 (3종): 불안정 서열, 순수 서열, 적응체
- 재해석 가능 (1종): 무속성
- 개념 유효 (1종): 기생체
